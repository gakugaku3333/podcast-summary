#2-02 「自分で決めること」は、幸福に直結する

起業家・つまもと氏とライター・泉氏が、人生における「意思決定の裁量」と幸福度の深い関係について語ります。リスクを取ることの本質や、現代社会における思考停止の罠、そして自分の限界（キャップ）を外すための考え方を提示します。

📌 リスクの本質は「意思決定の裁量」にある

  • リスクと幸福の相関：起業などの「リスクが高い」とされる行為に幸福を見出すのは、それが「人生のあらゆるイベントの意思決定権を100%自分で持っている状態」だからである。
  • 責任の範囲：自分で決めることは、自分で責任を取る範囲が広がることを意味する。この「責任の重さ」こそがリスクの正体だが、その裁量の幅が広いほど幸福を感じるタイプの人もいる。
  • 成長フェーズによる変化：子供の頃は親や先生に委ねていた意思決定を、成長とともにどれだけ自分の手に取り戻していくかが人生のプロセスである。

📌 現代社会における「思考停止」の罠

  • 社会の成熟と副作用：日本は平和で安全なため、強烈な意思決定をしなくても「流れに沿って」生きていけてしまう。これが「思考停止」を招き、本質的な幸せから遠ざかる要因になっている。
  • 生存本能の減退：かつては「何を食べるか」「どこへ行くか」を決めなければ死に直結していたが、現代はその必要がなくなった。
  • 常識を疑うマインド：社会が成熟しているからこそ、意識的に「本当にこれは正しいのか？」と常識を疑い、自分で選択する姿勢を持たないと、人生の主導権を失ってしまう。

📌 自分の限界（キャップ）をどう捉えるか

  • 能力の開放：多くの人が「自分はこんなものか」と上限を決めてしまっているが、それは現在の能力に基づいた誤解である。
  • 時間軸による変化：
    • 1年：かなりのことができる。
    • 2年：人生が大きく変わっていく。
    • 10年：全く違う人生にすることができる。
  • 正しい努力の重要性：方向性を見誤らなければ、人間は無限に成長できる。その「試行錯誤のプロセス」自体に幸福が宿る。

図：正しい努力が新しい景色を見せ、自己のキャップを更新し続けるサイクル

📌 「淡々（たんたん）」と積み上げる強さ

  • スーパースパイラルの到来：大谷翔平選手や井上尚弥選手のように、劇的な物語（逆境からの逆転など）よりも、圧倒的なルーティンを「淡々と」こなして偉大な記録を打ち立てるスタイルが現代の象徴。
  • 迷いのなさ：才能、やりたいこと、正しい練習方法が完全に合致した状態。外部からは淡々と見えるが、その内側では凄まじい集中力と努力が継続されている。
  • 競技レベルの向上：あらゆる分野で「正しい努力の方法」が一般化された結果、このスパイラルに入った者だけが圧倒的な高みに到達できる。

📌 意思決定と「愚痴」の関係

状態 結果に対する反応 マインドセット
自分で決めた場合 失敗しても「失敗談」になる 「次はこうしよう」という前向きな姿勢
他人に決められた場合 不満が「愚痴」になる 「あいつが決めたから」という責任転嫁
