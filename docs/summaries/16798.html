日本一たのしい哲学ラジオ：西田幾多郎シリーズ 第1回

「日本最初の哲学者」と称される西田幾多郎を紐解く新シリーズの導入回。明治維新から第二次世界大戦終結直前までという激動の時代を生き抜いた彼の生涯、思想、そして現代において彼を学ぶ意義について、パーソナリティの品川氏が熱量を持って語ります。

西田幾多郎 京都学派 善の研究 純粋経験 哲学の道 明治維新 禅 浄土真宗 自覚 場所の論理

📌 西田幾多郎とはどのような人物か

  • 日本を代表する哲学者：世界的な哲学史に名を残す、日本で最も有名な哲学者の一人。
  • 西洋と東洋の融合：明治以降に輸入された西洋哲学を咀嚼し、東洋的な直感や「禅」の思想と融合させた。
  • 京都大学（京都帝国大学）教授：京都学派の創始者。京都の観光名所「哲学の道」は、彼らが思索しながら歩いたことに由来する。
  • 哲学者としての姿勢：常識とは異なる世界像を直感的に捉え、それを一生かけて言語化しようと試み続けた人物。

📌 シリーズで注目すべき「日常語」の再定義

西田哲学を学ぶと、日常的に使っている言葉が全く異なる特別な意味（西田的意味）を帯びて聞こえるようになる。以下のキーワードが今後のシリーズで重要となる。

キーワード 西田哲学における重要性
純粋・経験 主客未分の状態を示す「純粋経験」として登場。
自由・働く 単なる動作ではなく、存在の根源に関わる「行為」として語られる。
自覚 西田が晩年まで考え続けた、自己を映し出す重要な概念。
場所 「場所の論理」として知られる、西田哲学の独創的な到達点。
絶対・矛盾 「絶対矛盾的自己同一」という有名なフレーズに繋がる。

📌 時代背景：激動の76年間

  • 生涯の要約：本人の言葉によれば「人生の前半は黒板を前に座り（学生）、後半は黒板を背にして立った（教師）。黒板の前で一回転しただけ」という極めてシンプルなもの。
  • 実際の内実：本人の言葉とは裏腹に、私生活では家族の死や経済的困窮など、凄まじい「流転」と「濁流」を経験している。

📌 家族構成と幼少期のルーツ

  • 父・安則：教育熱心だが気性が荒く、事業（米相場など）に失敗して家を没落させた。西田の結婚生活に干渉し、無理やり離婚させるなどのエピソードも残る。
  • 母・とさ：誠実で勤勉、篤い仏教（浄土真宗）信者。西田が最も慕い、思想的にも影響を受けた人物。
    • 5歳の西田に「蓮如上人の御文（おふみ）を読んだらおっぱいをあげる」と教え、西田はそれをスラスラ読んでいたという。
  • 兄弟の悲劇：
    • 最も親しかった姉を14歳の時にチフスで亡くす。
    • 弟の兵次郎は日露戦争で戦死。
    • 自身の子供たちも多くが先立つなど、常に「死」や「悲しみ」が身近にある人生だった。

📌 今後のシリーズ構成

西田幾多郎の人生と思想を、以下の四部構成で展開していく予定。

構成 テーマ 内容のイメージ
第一部：流転（るてん） 苦難の青年期 家族の不幸や学業での挫折、定まらない職。
第二部：頭角（とうかく） 思想の芽生え 『善の研究』の執筆と世間への登場。
第三部：躍進（やくしん） 京都学派の形成 独自の哲学体系を構築し、日本哲学の頂点へ。
第四部：濁流（だくりゅう） 晩年と戦争 戦争の足音と、極限状態での思索の深化。
「西田の言っていることの9割は理解できないかもしれない。しかし、彼が見た世界の1%でも言葉を通じて重なり合えたとき、日常の景色が変わって見えるはずだ。」（品川）
