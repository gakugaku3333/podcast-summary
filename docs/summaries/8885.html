📌 労働の本質：第4章「154節」の深い洞察

  • 単なる生計の手段ではない：労働は、お金を稼ぐためだけのものではない。
  • 3つの重要な側面：
    • 自己表現：自分の能力や意志を形にする場。
    • 人間関係：他者と繋がり、交流する場。
    • 地域社会への貢献：誰かの役に立っていると実感する場。
  • AIによる崩壊の危機：効率化だけを追求すると、これらの「人間的な側面」が削ぎ落とされてしまう。

📌 人間の幸せを構成する「4つの要素」

日本理化学工業の大山会長の言葉を引用し、働くことがいかに人間の尊厳に直結するかを整理しています。

  • 1. 人に愛されること
  • 2. 人に褒められること（★仕事から得られる）
  • 3. 人の役に立つこと（★仕事から得られる）
  • 4. 人から必要とされること（★仕事から得られる）
「愛されること以外の3つは、仕事を通じて得られるもの。だからこそ、重度の障害を持つ方であっても、働く場を提供することがその人の幸せに繋がる。」

📌 テクノロジーが「ありがとう」を取り戻す

  • 「すいません」から「ありがとう」へ：
    • 介助を受けるだけの状態では、申し訳なさから「すいません」が口癖になり、社会との繋がりを断ってしまう。
    • オリィ研究所（分身ロボットOriHime）の例：体が動かなくても、ロボットを通じて接客などの仕事をすることで、客から「ありがとう」と言われる側になれる。
  • AIの役割：持たざる人、弱き人に「階段（手段）」を提供し、他者と補完し合える関係を築くためのツールであるべき。
  • プラスサムのゲーム：誰かが独占するのではなく、お互いに「お願い」し、「感謝」し合うことで、社会全体の価値を高めていく。

📌 結論：AI時代に求められる「思想」と「実践」

  • 機能的価値を超えて：AIに「やる気のない人」が代替されるという冷徹な視点だけでなく、場としての職場の価値（人間関係や社会的立場）をどう守るかが問われている。
  • 隣人への愛：目の前の人のために何ができるか、という具体的な実践が共同体の豊かさを作る。
  • 思想の重要性：AIの技術的知識だけでなく、宗教や歴史が積み重ねてきた「人間とは何か」という思想を学び、実践する力がこれまで以上に価値を持つ。
