【高橋祥子】善人だけが集まってもいい社会はできない『社会はどう進化するのか』

生命科学研究者の高橋祥子氏をゲストに迎え、進化生物学者デイビッド・スローン・ウィルソンによる著書『社会はどう進化するのか』を紐解くエピソード。個人の善意や能力ではなく、集団の「設計」がいかに社会の成否を決めるのか、進化論の視点から解説します。

📌 ゲスト・高橋祥子氏の現在の活動

  • 新会社「LAS」の設立：2023年にジーンクエストの社長を退任し、老化・長寿（ロンジェビティ）を研究するバイオテック企業を立ち上げ。
  • 老化の定義：見た目の問題ではなく、科学的には「病気になりやすい状態」を指す。老化を予防することで、健康なまま生涯を終える人を増やす社会実装を目指す。
  • XPRIZE Healthspanへの挑戦：
    • 米国のエクスプライズ財団が主催する、健康寿命を10年若返らせることを目指す賞金150億円のサイエンスコンペ。
    • 世界600チーム以上の中から、現在セミファイナルの100チームに選出されている（日本からは10チーム未満）。
    • 2030年のファイナルに向けて研究を継続中。目的は優勝だけでなく、研究コミュニティの活性化にある。

📌 紹介書籍：『社会はどう進化するのか』

  • 著者：デイビッド・スローン・ウィルソン（進化生物学者）。
  • テーマ：社会全体の設計論を進化生物学の視点から論じた本。
  • 核心的な主張：社会がうまくいかないのは「人間が愚かだから」ではなく、「集団のルールや構造が協力を維持できる設計になっていないから」である。
  • 表紙の意図：蜂の巣と無数の蜂が描かれている。蜂の社会がなぜ高度な協力を維持できているのかという事例が、人間社会への示唆として重要。

📌 進化から見る「良い集団」の条件

  • 個別最適 vs 全体最適：
    • 癌細胞の例：細胞個体として「増殖したい」という欲求（個別最適）に従うと、最終的に個体（全体）を殺し、自分たちも死滅する。
    • 個人が自分の生存可能性を上げる行動を無条件に取っても、自然に良い社会ができることはあり得ない。
  • 進化は「競争」ではなく「協力」を生む：
    • 一般的な進化論のイメージ（弱肉強食・個体間の競争）とは異なり、実際には「協力できる集団同士の競争」が起きている。
    • 内部で足を引っ張り合う集団は淘汰され、うまく協力できる集団が生き残る。これが「進化」の本質。
  • 成功するコミュニティの共通点：
    • 目的が明確であること。
    • アイデンティティ（境界線）が強いこと。
    • 裏切り（フリーライダー）を排除・抑制できる仕組みがあること。

📌 前回の話題（ジョナサン・ハイト）との共通点

  • 感情と理性の役割：
    • 前回の本（『社会はなぜ左と右にわかれるのか』）では「感情が先にあり、理性が後付けされる」と説かれた。
    • 人間の感情は、個体や家族などの小さな集団を生き残らせるためには有利に働く。
    • しかし、その感情だけでは、国家や現代社会のような巨大な集合体において「全体最適」を実現することはできない。
  • 結論の合致：どちらの本も「放っておいても良い社会はできない」という点で一致している。

📌 進化論を社会に応用する意義

  • 適応と進歩の違い：進化は単なる「環境への適応」であり、必ずしも「進歩」ではない。
  • 現代の課題：人間が生物として進化の過程で獲得した機能が、必ずしも現代や未来の複雑な社会に適応可能であるとは限らない。
  • 設計の重要性：人類がいまだに戦争を繰り返すのは、知能の問題ではなく、集団としての進化・設計が追いついていない側面がある。
