南シナ海をめぐる14カ国共同声明と地政学的リスク

2026年7月14日放送。南シナ海における中国の海洋権益主張に対し、日本を含む14カ国が共同声明を発表したニュースを中心に、ホルムズ海峡の緊張や海上交通路（チョークポイント）の経済的重要性を解説しています。

南シナ海 共同声明 仲裁裁判所 ホルムズ海峡 チョークポイント 台湾海峡 マラッカ海峡 CSIS 地政学リスク

📌 ホルムズ海峡の最新情勢

  • アメリカとイランの対立激化：12日にアメリカがイランの軍事施設など140カ所を攻撃。
  • イランの対抗措置：革命防衛隊がホルムズ海峡の封鎖を宣言。
  • 海運への影響：停戦合意への期待もあったが、保険料の高騰、機雷への警戒、武装勢力の攻撃リスクにより、各海運会社は慎重な姿勢を継続。

📌 南シナ海に関する14カ国共同声明

  • 参加国：日本、フィリピン、アメリカを含むアジア太平洋およびヨーロッパの計14カ国。
  • 声明の核心：中国の拡張的な海洋権益に関する主張には「法的根拠がない」とする2016年の仲裁裁判所の判断を再確認。
  • 中国の反応：
    • 即座に反論し、南シナ海の権限を改めて強調。
    • 緊張の原因は「アメリカを含む外部勢力による軍事活動」であると主張。
    • 北京の日本大使館公使を呼び出し、「日本に口出しする資格はない」と抗議。

📌 歴史的背景：2016年の仲裁裁判判決

  • 提訴の経緯：2013年、フィリピンが中国によるスカボロー礁の実行支配や人工島造成を受け、ハーグの仲裁裁判所に提訴。
  • 判決内容（2016年7月12日）：中国が南シナ海ほぼ全域に主権を持つという主張には、国際法上の根拠がないと認定。
  • 中国の立場：当初から手続きへの参加を拒否。判決を「違法で無効な紙くず」として一貫して受け入れを拒否している。
  • 現状：判決から10年が経過したが、中国は「南シナ海に公海は存在しない」と主張するなど、緊張はさらに高まっている。

📌 経済的生命線としてのチョークポイント

南シナ海周辺の海峡は、世界の物流が集中する極めて重要な「チョークポイント」です。

主要な統計データ（2024年予測）

項目 詳細・数値
南シナ海全体の通過物資額 6兆4,000億ドル
マラッカ・台湾海峡のシェア 世界の海上貿易の約21%
日本の貿易依存度（マラッカ） 18%（主に中東からの石油・天然ガス）
日本の貿易依存度（台湾海峡） 28%（主に半導体などの技術関連）
中国の貿易依存度（台湾海峡） 輸入の33%、輸出の16%

📌 中国が抱えるジレンマ

  • 自国経済への打撃：台湾海峡で有事が起きれば、中国自身の輸出入（特に輸入の33%）が遮断され、経済に甚大な被害が出る。
  • 構造的矛盾：自国の軍事行動が、自国経済の生命線を脅かしかねないというジレンマを抱えている。
  • 今後の注視点：この経済的リスクを考慮した上で、中国が今後どのような強硬策、あるいは抑制策をとるのかが焦点となる。
南シナ海や台湾海峡における航行の自由は、日本を含む世界経済にとって不可欠な要素であり、その遮断は計り知れない影響を及ぼす。
