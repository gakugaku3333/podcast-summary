超相対性理論 #256 今を生きるための伝統とは何か？

発酵デザイナーの小倉ヒラクさんをゲストに迎え、新著「僕たちは伝統とどう生きるか」を軸に、現代における伝統の意義や継承のあり方について探求するエピソードです。伝統を単なる「守るべき古いもの」としてではなく、現代を生きるためのツールとして捉え直す議論が展開されます。

📌 新著「僕たちは伝統とどう生きるか」の背景

  • 執筆の動機：伝統産業が直面する「複合的なピンチ」
    • 気候変動：魚が獲れなくなり、魚醤が作れない。
    • 後継者不足：週に数袋しか生産できない現状。
    • インフラの崩壊：古い機械が壊れても、修理できるメーカーが既に存在しない。
  • 本の狙い
    • 「伝統を守ろう」という既存の啓蒙的なアプローチではなく、伝統と直接関係のない人々が「今を生きるために伝統をどう使えるか」を提示する。
    • 各地で物作りを続ける友人たちが、なぜ大変な状況でも作り続けるのか、その潜在的な意味を原理から探る。
  • 反響
    • 発売前にバリューブックスでの先行予約だけで2000部を完売。
    • 出版初日に重版が決定。
    • 実用書や産業論としてだけでなく、思想書として読まれている。

📌 伝統の二面性：大文字と小文字

本の中で提示される、伝統を捉えるための重要な補助線です。

概念 特徴 関連するキーワード
大文字の伝統 権威的、固定的、公的な記録 国家、血統、文字記録、一等星
小文字の伝統 身体的、流動的、日常的な実践 伝承、行為、手の中に宿るもの、六等星
  • 知識と知恵（ティム・インゴルドの引用）
    • 知識：対象を固定化し、安定させるもの（大文字に近い）。
    • 知恵：現実を柔軟に受け止め、足の裏で感覚を掴みながら歩く行為（小文字に近い）。

📌 継承における「ままならなさ」と「突然変異」

  • 模倣の失敗としての創作
    • 渡辺氏の持論「あらゆる創作は模倣の失敗である」と、伝統の継承は響き合う。
    • 完璧にコピーしようとしても、時代背景や個人の資質の違いにより、必ずズレが生じる。
  • 「昭和元禄落語心中」からのインスピレーション
    • 師匠の芸をそのまま継ぐのではなく、失われた別の芸の面影を追いかけたり、自分なりの解釈を加えたりする中で、新しい芸が生まれる。
    • 「理想的な伝承」が起こらないというままならなさの中にこそ、物語と進化が宿る。
  • デカルトの例（森田真生氏の説）
    • デカルトは古代ギリシャ哲学を現代に完全再現しようとした。
    • しかし、時代が違うため、完全再現を試みた結果、全く新しい哲学（クリエイション）になってしまった。
