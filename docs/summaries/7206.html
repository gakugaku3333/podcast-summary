超相対性理論 #257：今を生きるための伝統とは何か？（ゲスト：小倉ヒラクさん）

発酵デザイナーの小倉ヒラク氏を迎え、新著「僕たちは伝統とどう生きるか」を深掘りする第2回。伝統を「守るべき結果」ではなく「生成されるプロセス」として捉え直し、現代における継承のあり方を、生命論や芸術、さらにはトロンボーンの和音に例えて考察します。

📌 伝統の本質：結果か、プロセスか

  • 伝統の定義の再構築：伝統を「固定された守るべきもの」と捉えるのではなく、その時々の人々の連なりや代謝（プロセス）そのものとして捉える。
  • 生命論との接続：
    • 結果としての生命（プラトン的）：一つの超越的な正解や形に還元しようとする考え方。伝統を「型通りに守る」姿勢に近い。
    • プロセスとしての生命（アリストテレス的）：分岐や派生、生成消滅を繰り返す流れそのもの。伝統を「使い、遊ぶ」姿勢に近い。
  • 俯瞰の否定：伝統の全体像を完璧に俯瞰し、ジャッジできる人間は存在しない。当事者がその時々で必死に繋いでいるプロセスの集積こそが伝統の正体である。

📌 「模倣の失敗」が個性を生む

  • 完コピの不可能性：師匠の技を完璧に再現しようとしても、身体差、素材の差、時代の状況差により、必ず「ズレ」が生じる。
  • 表現とは模倣の失敗である：
    • 意図的に革新しようとしなくても、やむにやまれず変わってしまう。
    • その「どうしても埋められない差分」こそが、次の世代の個性や新たな伝統の芽となる。
  • ガダマーの了解：過去を完全に理解することは不可能であると認め、その隔たりを無理に埋めようとしないことが、継承のスタート地点となる。

伝統継承における「意図せぬ変化」の構造

📌 大文字の伝統と小文字の伝統

概念 特徴 役割・ニュアンス
大文字の伝統 (Tradition) 権威、ブランド、固定的なイメージ マーケティングやブランディング、安心感の提供
小文字の伝統 (tradition) 個人の実践、微細な変化、遊び 現場のリアリティ、生命力のある代謝
  • 対立ではなくレイヤー：両者は単純な対立構造ではなく、重なり合っている。
  • 戦略的使い分け：危機的な状況では「小文字」をあえて「大文字」化して守る戦略もあれば、強すぎる「大文字」の中に「小文字」の微細な変化を読み取る感性も重要。

📌 「変わらない」と感じさせるための高度な調整

  • トロンボーンの比喩：
    • ピアノ（デジタルな固定音）と違い、トロンボーンは無段階で音を調整できる。
    • その場の空気、湿度、他の楽器とのズレを微調整し続けることで、初めて「完璧な和音（神の権現）」が生まれる。
    • 伝統も同様：時代や受け手の変化に合わせて「微調整（イコライズ）」し続けるからこそ、受け手は「変わらない懐かしさ」を感じることができる。
  • バージル・アブローの「3%ルール」：
    • 既存のデザインを97%継承し、3%だけ差分を加える。
    • 50%変えると別物になり、0%だと単なるコピーになる。3%という「ごくわずかな逸脱」こそが、同じ系統であることを認識させつつ新しさを生む。
  • 藤原定家の「本歌取り」：
    • 過去の名歌の言葉を借りつつ、主題をずらす。
    • 認識できるギリギリの範囲で意味をずらす高度なルール設定。
「全く同じものを作り続けていたら、周りが変わっていく中で、逆に『変わってしまった』と感じられてしまう。変わり続ける世界の中で『変わらない』という価値を維持するためには、実は作り手側が文脈に合わせて変化し続けなければならない。」
