アダム・スミス編 第4回：『国富論』が解き明かした豊かさの正体

アダム・スミスの主著の一つ『国富論』をテーマに、経済学の基礎となった「分業」や「見えざる手」の真意を解説するエピソードです。一般的に誤解されがちな用語の定義を正しつつ、スミスが観察を通じて見出した人間社会の仕組みを紐解きます。

アダム・スミス 国富論 分業 見えざる手 交換性向 自己愛 重商主義 植民地主義批判 生産性

📌 資本主義のエンジン「分業」の驚異

  • 観察の人、アダム・スミス：スミスはアリストテレスのように現実を鋭く観察する学者であり、近所の「ピン工場」の観察から『国富論』を書き始めた。
  • 生産性の爆発的向上：
    • 1人の職人が全工程を行うと、1日に1本〜20本程度しか作れない。
    • 工程を18に細分化（針金を切る、尖らせる、頭をつける等）し、12人で分業すると1日48,000本ものピンが製造可能になる。
    • 生産性は200倍〜4800倍に跳ね上がる。
  • 分業の2つの側面：
    • 作業場内の分業：工場内での生産プロセスの細分化。
    • 社会的な分業：個々人が自分の得意なことや儲かることに特化し、社会全体で役割を分担すること。

図：分業が生産性を高めるメカニズム

📌 人間の本能「交換性向」

  • 人間特有の性質：他の動物（例えば犬）は、自分の利益のために骨を交換しようとはしない。人間だけが「自分の利益のために交換する」という本能（交換性向）を持つ。
  • 互恵性の発見：自分の利益を追求して交換することが、結果として相手の利益にもなる。これが社会的分業を成立させる基礎となっている。
  • 自己愛の肯定：
    「私たちが食事ができるのは、肉屋や酒屋やパン屋の慈悲心によるのではなく、彼らが自分自身の利益を追求しているからである。」
  • キャリア論への繋がり：身分に縛られず、自分の得意や好きを活かして仕事を選べるのは、この社会的分業と交換の仕組みがあるからこそ。

📌 「見えざる手」の真実と誤解

  • 「神の」は付かない：一般的に「神の見えざる手」と言われるが、スミス自身は「見えざる手（an invisible hand）」としか書いていない。
  • 限定的な登場：膨大な『国富論』の中で、この言葉が登場するのはわずか1箇所のみ。
  • 本来の意味：
    • 個々の投資家が、自分の資産の安全と利益を最大化しようとして「国内の産業」を支援する。
    • その意図はないものの、結果として社会全体の年間の収入（国富）を最大化することに繋がる。
    • 「個人の利益追求が、意図せず公共の利益を促進する」という市場の調整機能を指している。

📌 市場がもたらす「道徳的教育」

  • 信頼がビジネスの基盤：自由な市場で継続的に利益を上げるためには、他人から信頼されなければならない。
  • 道徳の陶冶（とうや）：
    • 嘘をつく者や不誠実な者は市場から淘汰される。
    • 特に中流以下の人々にとって、市場への参画は「わがままを抑え、道徳的に振る舞う」ための教育的な場として機能する。

📌 「富」の再定義と植民地主義への批判

概念 重商主義（当時の主流） アダム・スミスの考え
富の定義 国が保有する金・銀の量 国民が消費できる商品・サービスの豊かさ
貿易の目的 輸出を増やし、金銀を蓄積する（黒字至上主義） 交換によって生活を便利にし、消費を豊かにする
植民地政策 資源確保のために拡大すべき 「愚行と不正」であり、維持コストが高い
  • 植民地主義への厳しい批判：
    • 金銀を求めて先住民の土地を奪う行為を「不正」と断じた。
    • アメリカの植民地支配についても、イギリスはコストをかけて維持するより、分離して自由な関係を築くべきだと主張した。
  • 啓蒙時代の知識人：当時の政府批判に近い内容も、理性的・観察的な視点から堂々と記述していた。
