SNS依存をめぐる裁判：メタとGoogleに初の損害賠償評決

米国ロサンゼルス州裁判所にて、未成年のSNS依存に対するプラットフォーム企業の過失責任を認める画期的な評決が下されました。これまで法的責任を免れてきた巨大テック企業に対し、サービスの「設計」そのものが中毒性を生んでいるとして賠償を命じた、歴史的な転換点となるニュースです。

📌 オーストラリアにおけるSNS規制の現状

  • 16歳未満のSNS利用禁止法：2023年12月に施行されたが、実効性に課題。
    • 施行後も多くの子どもがアカウントを持ち続けていることが調査で判明。
    • 年齢確認の方法に「抜け穴」が存在している。
    • 当局は運営会社の対応について調査を開始。

📌 米国での損害賠償評決の概要

  • 裁判の当事者：
    • 原告：現在20歳の女性（6歳でYouTube、9歳でInstagramを開始）。
    • 被告：メタ（Instagram運営）、グーグル（YouTube運営）。
  • 被害の内容：SNSへの依存により、うつ病などの深刻なメンタルヘルスの不調を来した。
  • 賠償命令額：合計 600万ドル（約9億5,000万円）
    企業名 賠償額
    メタ (Instagram) 420万ドル
    グーグル (YouTube) 180万ドル

📌 裁判の争点：コンテンツから「設計」へ

今回の裁判が重要視されている最大の理由は、訴えの論点が「投稿内容」ではなく「アプリの仕組み（設計）」にある点です。

  • 中毒性の指摘：無限スクロールやおすすめ投稿のアルゴリズムが、タバコやギャンブルのような中毒性を意図的に持たせていると主張。
  • 警告の欠如：プラットフォーム側がその危険性についてユーザーに適切に警告していなかった点が過失とされた。

📌 今後の影響と歴史的背景

  • 数千件の先行事例：現在、米国では未成年の保護者らによる同様の訴訟が数千件以上あり、今回の評決がそれらの行方を左右する。
  • 「タバコ訴訟」との類似性：
    • 1990年代、タバコ業界が中毒性を認識しながら安全対策を怠ったとして巨額の賠償を命じられた事例になぞらえられている。
    • 将来的にSNSでも、広告規制やパッケージのような「警告表示」が義務付けられる可能性がある。
  • 企業の対応：メタとグーグルは不服として上訴する方針。裁判は長期化が予想される。
  • 根本的な変更の可能性：消費者の声が高まれば、企業はサービス設計そのものの変更を避けられなくなる。

📌 考察：音声メディアの役割

  • スクリーンタイムの葛藤：教育的・便利な動画コンテンツは多いが、親世代には「これ以上画面を見せたくない」という強い抵抗感がある。
  • 音声媒体の優位性：耳だけで情報を伝える音声メディアは、視覚的な依存を避けつつ情報を届けられる「安心できる方法」として再評価されている。
