探究学習のつくり方：生徒の「やらされ感」をなくすには

タクトピアのラーニングデザイナー渡辺信也氏をゲストに迎え、探究学習において生徒が主体性を失い「やらされている」と感じてしまうメカニズムとその解消法について、発達段階や大人の関わり方の視点から深掘りします。

タクトピア ラーニングデザイナー やらされ感 フロー状態 意味づけ 自己決定 アントレプレナーシップ教育 緊張と緩和 非日常

📌 ゲスト紹介：渡辺 信也（しんや）氏

  • バックグラウンド：日本と韓国のハーフ。高校卒業まで香港で育ち、アメリカの大学院へ進学。
  • キャリア：2018年に来日し、タクトピアに入社。日本在住歴とタクトピア歴が同じ（約8年）ベテランラーニングデザイナー。
  • 専門性：高校生の頃から学校外での教育プログラム提供に携わり続けてきた。

📌 発達段階による「やらされ感」の違い

  • 小・中学生：
    • 活動そのものに夢中になる「フロー状態」に入りやすい。
    • 「なぜやるのか」という深い意味づけがなくても、目の前の作業（例：ことわざを100枚調べる等）自体を楽しめる傾向がある。
  • 高校生：
    • 認知能力が高まり、「大人の意図」を察知するようになる。
    • 大人がやらせたいことと、自分がやりたいことの乖離に敏感になり、反抗心や「やらされ感」が生じやすい。

📌 「やらされ感」を打破する3つのアプローチ

1. 徹底した「意味づけ」と場のセッティング

  • 非日常の演出：外部の人間（タクトピア等）が学校に入ることで、日常の「当たり前」を揺さぶり、説明が必要な状況を作る。
  • 多層的な説明：
    • 全体レイヤー：プログラム全体の目的。
    • ワークレイヤー：今このアクティビティをやる理由。
    • 個人レイヤー：これをやることで自分にどんな得があるか。

2. プロジェクト進行中の働きかけ

  • 中だるみを許容する：モチベーションの波は自然なもの。大人が停滞を恐れすぎないことが重要。
  • 緊張と緩和の設計：中間発表や外部ゲストの招聘など、リズムを作ることで「息継ぎ」をさせる。
  • 選択の支援：大人が答えを指し示さず、生徒が自ら選べるように支援する。

3. 大人の「正解」を押し付けない態度

  • 自己決定の尊重：たとえ間違っていても「自分で考えたものをやりたい」という思いを優先させる。
  • 想像力の保持：大人の経験則で「無理だ」と断定せず、10年後の可能性などを一緒に面白がる。

📌 象徴的なエピソード

ケース 内容 学び・視点
韓国でのアントレ教育 大人向けメンターが高校生に「そのソリューションは絶対うまくいかない」と厳しく指導した。 生徒は「自分がやりたいことをやるのがアントレなのに、これでは違う」と違和感を抱いた。
ドローン勉強部屋 中学生が「地域に勉強場所がない」課題に対し、空飛ぶドローン部屋を提案した。 大人が否定せず「今は難しいが将来はあり得る」と返したことで、生徒は自ら「移動バス」案へ着地させた。
先生向け研修 広島の学校で、先生自身が探究（試作・実施）を行う研修を実施。 先生も「分からない問い」に挑む背中を見せることで、生徒との共通基盤（仲間意識）が生まれる。

📌 結論：大人が持つべきアイデンティティ

  • コンサルタントでも先生でもない：「教育の機会を作っている人」という意識。
  • 分からないを大事にする：大人が「分からない状態」を許容し、未知のものに触れ続ける姿勢が、生徒との共創を生む。
  • 余白を残す：大人が理論武装して正解を導くのではなく、生徒が自己決定できる「余白」をどれだけ我慢して作れるかが鍵。
「大人が正解を正すのではなく、そうじゃない未来を大人側が描けるかどうかが、子供たちの自己決定に作用する」
