超相対性理論 #258 今を生きるための伝統とは何か？（その3）

発酵デザイナーの小倉ヒラク氏をゲストに迎え、伝統を「守るべき固定物」ではなく「変化し続けるプロセス」として捉え直す議論の第3回。微細な変化の積み重ねがもたらす創造性や、組織運営における「小文字の伝統」のあり方について深掘りします。

📌 伝統における「変化の度合い」と3%ルール

  • 3%のクリエイティビティ：ナイキの「3%ルール（97%は変えず、3%だけ新しくする）」を引用し、伝統の継承もわずかな差分の積み重ねであると指摘。
  • DNAレベルの差異：人間の遺伝子も0.1%〜0.7%変わるだけで別物になる。伝統もそれほど微細な変化（アローアンス）の中で動いている。
  • 言語の変遷：
    • 女子高生言葉などの若者言葉は、日本語という枠組みの99%を維持した上での1%の遊び。
    • その1%の遊びが数世代積み重なると、明治時代の日本語が現代人には読めなくなるほどの大きな変化（断絶）を生む。
  • 比較による差分の顕在化：各地の味噌を並べることで、共通点（97%）の中にある地域ごとの差異（3%）が際立ち、伝統の個性が認識される。

📌 生命の本質：意識派 vs 現象派

小倉ヒラク氏は、伝統の捉え方を「生命とは何か」という問いに重ねて整理しています。

立場 主な思想家 特徴
意識派 プラトン、ハイデッカー 人間は「意識」である。世界を意識の中に閉じ込め、コントロールしようとする「世界へのマウント」。
現象派 フランシスコ・バレラ 人間は「現象」である。バクテリアや菌と同列。意識は付随的なものに過ぎない。
  • 主体性の手放し：発酵のプロセスに関わる際、自分の主体性や自由意志を適度に手放さないと、発酵の本質や「美味しさ」を理解できない。
  • デュアルな視点：現代人は「意識派」を捨てきれないが、97%の意識（ハイデッカー的）の中に3%の現象（バレラ的）を混ぜるようなバランスが重要。

📌 組織運営における「小文字の伝統」

  • 標準化の拒否：発酵デパートメントでは、スタッフ教育において業務の完全なマニュアル化を避けている。
  • ナラティブと緊急事態：
    • 伝えるのは「なぜこの店が存在するのか」というナラティブ（物語）と、テロや災害時の緊急事態マニュアルのみ。
    • 中間にある日常業務は現場の裁量に任せることで、個別の「小文字の伝統」が自然発生する。
  • 最小構成要素の模索：何%まで変わっても「その組織らしさ」が維持されるのか。20%の核が残っていれば、80%が変わっても同一性は保たれるという仮説。

📌 誤読とエラーがもたらす創造性

  • 倉田隆史氏の事例：ハイデッカーの予備知識がないまま本を読み、主流の「存在論」ではなく「道具論」に注目して民藝の文脈で解釈した。
  • 誤読の価値：正解をなぞるのではなく、自分勝手に引き寄せて「間違えて読んでしまう」ことの中に、組織の変革や新しい文化の源泉がある。
  • エラーを前提とした継承：
    • 生命がコピーミス（変異）を繰り返しながら適応するように、伝統も「正しく伝わらないこと」を前提とする方が健全である。
    • 真剣に自分事として捉えるほど、既存の正解からズレた「謎の読み」が発生し、それがクリエイティビティになる。

📌 連続性のパラドックス：テセウスの船

  • 物理的連続性と概念的連続性：
    • テセウスの船：部品がすべて入れ替わった船は、元の船と同じか？
    • マンチェスター・ユナイテッド：150年の歴史の中で、監督、選手、オーナー、ファンが全員入れ替わっても「同じチーム」として存続している。
  • 流れとしての存在：鴨長明の「ゆく河の流れは絶えずして、しかももとの水にあらず」のように、構成要素（水分子）ではなく「流れ（プロセス）」そのものが存在を定義する。
