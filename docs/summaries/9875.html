深津貴之のGUILD TALK：AI時代の幸福論（ゲスト：尾原和啓）

AIが当たり前になった社会において、人間がどのように生き方や幸福を定義し、生存戦略を立てるべきかを深津貴之氏と尾原和啓氏が語り合います。個人のサバイバル術から、マクロな社会構造、幸福の感じ方のチューニングまで、多岐にわたる知見が整理されています。

📌 AI時代の生存戦略：信頼とビジビリティ

  • 「5人の法則」によるリスク分散
    • 「尾原と仕事がしたい」という人を常に5人以上確保しておく。
    • 1人を失っても収入減は20%に抑えられ、その間に次を探せる。
    • AI時代だからこそ、代替不可能な「信頼」と「通じ合える関係性」が重要。
  • AIビジビリティ（AIへの露出度）
    • AIが推薦エンジンとなる時代、AIに「この金額でこの人を雇えるのはお得」と言わせる状態を作る。
    • 特定のコンテキスト（文脈）において、AIが提示する3〜5つの選択肢に必ず入るブランド力を構築する。
    • 毎月、複数のAIに自分やライバルの評価を問い、自身の立ち位置をピボット（方向修正）する。

📌 マクロ社会への視点：他人の善意に依存しない

  • ハイパーベーシックインカムの不確実性
    • イーロン・マスクらが提唱する「AIとロボットが富を生み、全員が裕福に暮らせる」未来。
    • しかし、その富の再分配は「資本家の胸三寸」や「国家の機能」に依存している。
  • 「他人の善意を前提としたギャンブル」を避ける
    • 企業の欲望（競争）が解決してくれる問題（AIの精度向上など）には乗っかってよい。
    • 一方で、再分配のような「善意」に依存する部分は、それが機能しなかった時のリスクを個人で持っておく必要がある。

📌 幸福論：リソース管理と「レンジ幅」

  • リソース論としての幸福
    • 不幸を排除するために、まずは低レイヤーのリソース（生存基盤）を埋める。
    • 高尚な悩みはリソースが充足して初めて向き合えるもの。
  • 幸福の「レンジ幅」を広げる
    • 10万円の高級料理も楽しめるが、300円の屋台飯やコンビニ飯も「うめえ」と心から思える感性を維持する。
    • 生活水準が上がって「これしか受け付けない」となるのは、一種の「味覚障害（選択肢の減少）」であり、ロバストネス（強靭性）を下げる。
    • 安宿や漫画喫茶でも快適に過ごせる「不幸にならない閾値」を低く保つことが、変化の激しい時代の自由につながる。
  • 自己定義の柔軟性
    • 「広い部屋にいるから偉い」といった外部報酬にアイデンティティを固定すると、環境変化で容易に崩壊する。
    • クソゲーをどう楽しむか、どうルールを変えるかという「探求」そのものに幸福を見出す。

📌 次世代の姿：ソーシャルとAIの融合

  • Z世代のナチュラルな社会意識
    • SNSで繋がっていることが前提の世代は、倫理観やコンプライアンスが内面化されている。
    • 「社会課題の解決」を自分ごととして捉え、AIをそのための「壁打ち相手」や「プロトタイプ作成ツール」として使いこなしている。
  • AIによる「自分らしさ」の深化
    • AIにアイディアを出させるだけでなく、実験と検証を繰り返すことで、より深い自分独自の知見に到達している若者が増えている。
    • ソーシャルな意識が先にあり、そこにAIという武器が加わったことで、地球規模でハッピーな構造が生まれつつある。

📌 結論：メタに生きることが「まっすぐ」な時代

  • メタスキルの重要性
    • かつて「メタに生きる（構造を俯瞰する）」ことは斜に構えたハックの一種だった。
    • 現代はAIによって誰もが構造をいじれるため、メタに生きることこそが最も誠実で「まっすぐ」な生き方になっている。
  • 自由の定義
    • 「自らを良しとする（自らを由とする）」。自分らしくあるための選択肢を持ち、その中から選べる状態こそが自由であり、幸福の土台となる。
尾原和啓氏のメッセージ：
「AI時代は、やったもん勝ち、試したもん勝ち。若さというAIで手に入らない武器を使い、目の前のメタな構造をちょっといじってみることを楽しんでほしい。」
