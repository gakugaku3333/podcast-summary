【5月21日】日韓首脳会談。アジアの「力の空白」を埋めるエネルギー安保戦略

2026年5月19日に韓国・安東（アンドン）で開催された日韓首脳会談の詳報。ウクライナ戦争や中東情勢の影響でアジアにおけるアメリカのプレゼンスが低下する中、日韓がエネルギー安全保障と防衛協力でどのように連携を深めるか、具体的な枠組み「パワーアジア」を中心に解説します。

📌 首脳会談の概要と背景

  • 開催日時・場所：2026年5月19日、韓国・安東（イ・ジェミョン大統領の地元）。
  • 会談の頻度：高市首相就任（昨年10月）以来、わずか半年余りで3回目の会談。
  • 外交スタイル：首脳同士が互いの故郷を行き来する「シャトル外交」が定着。
  • 最大のテーマ：「アジアの空白リスク」への対応。
    • アメリカが中東情勢への対応に注力しているため、アジアにおけるプレゼンスが低下。
    • ウクライナ戦争や中東情勢の緊迫化に伴う地政学的リスクへの共同対処。

📌 エネルギー安全保障の新枠組み「パワーアジア」

  • ASEK（アジアゼロエミッション協同体）の進化：
    • 日本が提唱したASEAN諸国・豪州との脱炭素枠組みを拡張。
    • 韓国などを加えた「ASEKプラス」として首脳会談を開催。
  • パワーアジア（Power Asia）構想：
    • 目的：アジア全体での協調したエネルギー（原油・LNG）確保。
    • 背景：韓国や東南アジアは日本に比べ石油備蓄量が少なく、価格高騰時の調達資金も不足しがち。
    • 支援内容：日本が約100億ドル規模の金融支援を実施。
    • 具体的施策：原油調達の円滑化、域内への備蓄タンク建設の後押し。
  • 日本国内への影響：高市首相は「国内需要への悪影響はなく、あくまでアジア諸国との協調確保である」と強調。

📌 防衛協力と地政学的対応

  • 北朝鮮・ロシアへの警戒：
    • ロシアと軍事協力を深める北朝鮮のミサイル能力向上への対処。
    • 日韓、および日米韓の3カ国で緊密に連携することを確認。
  • 現実路線の選択：
    • 歴史的背景による摩擦はあるものの、目の前の脅威に対処するため「現実路線」での歩み寄りを優先。
  • ホルムズ海峡：事態沈静化に向けた相互努力の確認。
