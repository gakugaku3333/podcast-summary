ながらAIラジオ：Manusインスタ連携、Elevenlabs V3、確定申告でのClaude Code活用

2026年3月12日収録。Meta傘下のManusによるInstagram連携や、音声生成AI Elevenlabsの最新モデルV3の衝撃、そして確定申告という実務におけるClaude Codeの具体的な活用事例について、AIコンサルタントのうすたく氏とメルカリAI推進担当の早川ごみ氏が語ります。

📌 ManusとInstagramの連携：ストーリー投稿の自動化へ

  • ManusがInstagramと連携：Metaが買収したAIエージェント「Manus」が、ついにInstagramでの操作に対応。
  • ストーリーの下書き作成：API制限の厳しいInstagramにおいて、外部からストーリーの下書きを作成できるようになった。
  • 完全自動投稿の現状：現時点では「パブリッシュ（公開）」ボタンを手動で押す必要がある環境が多いが、定期タスク実行による予約投稿に近い運用が可能になりつつある。
  • 運用の変化：これまで手動での「ちまちました更新」が必要だったInstagram運用が、AIによって効率化される兆し。

📌 Metaの買収戦略とAIエージェントの動向

  • Moltbookの買収：AIエージェントが自律的に投稿・交流するSNS「Moltbook」をMetaが買収。
  • 人材・データ争奪戦：OpenAIに引き抜かれた優秀な人材（Peter等）に対抗するため、MetaはManusやMoltbookといったプレイヤーを次々と傘下に収めている。
  • パーソナルアシスタントの未来：24時間稼働し、ユーザーの代わりにSNSを運用したり情報を整理したりする「エージェント版ソーシャルグラフ」の構築を狙っている可能性。

📌 AIリテラシー向上：プログラミング的思考の重要性

  • Claude Code導入の壁：CLI（ターミナル）版の方がセットアップが意外と楽な場合もあるが、初心者には依然としてハードルが高い。
  • 学ぶべき概念：AIエージェントを使いこなすには、コードが書けなくても以下の概念を理解していると指示の精度が劇的に上がる。
    • Git/GitHub：バージョン管理（セーブポイント）の概念。
    • 制御構文：IF（条件分岐）、FOR（繰り返し）などの論理構造。
    • データ構造：JSON、YAML、Markdownなどの形式。
  • 推奨学習ツール：
    • Progate：基礎的な論理構造を学ぶのに最適。
    • CS50（ハーバード大学）：YouTubeで無料公開されているコンピュータサイエンスの入門講義。

📌 Elevenlabs V3：音声生成の新たな到達点

音声クローニングとナレーションの質が大幅に向上。V2とV3の比較は以下の通り。

モデル 特徴 評価
V2 声の質（音色）は本人に非常に近い。 喋り方やイントネーションに不自然さが残る。
V3 イントネーション、発音、リズムが完璧に自然。 声の似具合よりも「人間としての自然さ」が圧倒的。
  • 活用アイデア：英語学習コンテンツ（イケボでの聞き流し）、ASMR、教育系ナレーションなど、人間が介在しなくても高いクオリティで提供可能。

📌 実務活用：確定申告・決算業務でのClaude Code

うすたく氏による、Claude Codeを用いた事務作業の自動化フロー。

  • ファイル名の一括変更：バラバラな名称の領収書ファイルを「日付_項目名_金額」の形式に一瞬でリネーム。
  • Gmailからの抽出：Google Workspace CLIと連携し、メール内の領収書とおぼしきデータを数百件単位で一括ダウンロード。
  • 教訓：エージェントを活用することで、年末に溜め込んだ作業も大幅に短縮可能だが、毎月の積み重ねが重要。
