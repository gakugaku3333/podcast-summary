#261 自分の肩書き、どう使ってますか？／ゲスト：室越龍之介さん（その2）

文化人類学者の室越龍之介さんを迎え、肩書きが持つ「社会的な仮面」と「個人の自意識」の葛藤を深掘りします。ビジネスと学問における肩書きの性質の違いや、集団への帰属意識、そして「ホームとフィールド」の関係性について、3人のパーソナリティが自身の経験を交えて対話します。

肩書き 文化人類学 中二病 教示的・通じてき マージナル（境界的） 帰属意識 小木沙耶香 和田夏美 ホームとフィールド

📌 肩書きを構成する3つのベクトル

ヤンヤン氏が提唱する、自分を名乗る際の3つの方向性についての整理。

  • マーケットの視点: 社会的な分かりやすさや信頼。例：「トヨタの社員」「弁護士」。思考停止できるほど強力な記号。
  • 個人の自我・欲求: 「私はこうありたい」という内面的な願望。分かりやすい肩書きを拒絶する「中二病」的な自意識もここに含まれる。
  • 関係性の視点: 目の前の相手との間でどうありたいか。文脈に応じて名乗りを変える動的な側面。

📌 ビジネスと学問における「肩書き」の性質

時間軸の捉え方によって、肩書きの使い方が異なるとヤンヤン氏は指摘する。

区分 性質 特徴
教示的（今ここ） ビジネス的 「今、何ができるか」という機能を重視。現場の混乱を避けるために分かりやすい肩書き（例：人類学者）を戦略的に使う。
通じてき（系譜） 人文学的 「誰から何を習ったか」という師弟関係や歴史的背景を重視。一人前でないという自意識から、安易に名乗ることを躊躇させる。

📌 人間をどう捉えるか：バイアスか価値か

小木沙耶香氏の著書を引用し、学問的態度と肩書きの関連性を考察。

  • 人間を「バイアスの源」と捉える（理系的）: 観察者の主観や偏見を排除し、客観性を追求する態度。
  • 人間を「価値の源」と捉える（文系的）: 特定の人間がその場にいたからこそ生まれた経験や解釈を研究の核とする態度。
  • 肩書きの機能とノイズ:
    • セミナー等では「機能（専門家）」としての匿名的な肩書きで十分。
    • 深い人間関係では、機能以外の「ノイズ（その人らしさ）」が重要になる。

📌 肩書きを超えた自己紹介：「名前のない遊び」

手話通訳者・和田夏美氏のアプローチに見る、真の自己の表出。

  • 問いかけの魔法: 「子供の頃にしていた、名前のない遊びは何ですか？」と聞く。
  • 身体的記憶の呼び起こし: 遊びを語る時、人は身振り手振りを交え、社会的な仮面を外した「個人」の顔になる。
  • 非言語的な情報: 決まった正解がない経験を語ることで、所属や肩書きよりも鮮明に「その人らしさ」が浮かび上がる。

📌 帰属意識とマージナル（境界的）な生存戦略

3人のパーソナリティが語る、集団との距離感。

  • ヤンヤン氏の「相対化」:
    • 一つの集団に深く帰属したくない。常に複数の所属を持ち、互いに相対化させていたい。
    • 「中心にいると死んでしまう」というセンサーがあり、意図的に距離を保つ。
  • 渡邉（コータロー）氏の「コウモリ的自意識」:
    • 組織の創業メンバーであり中心人物に見えるが、内実としては「何かを代表している」感覚が希薄。
    • どこにいても自分は端っこにいる存在（マージナル）であるという認識が根本にある。
  • 樋口氏の「仲間作りの苦手意識」:
    • ホモサピエンスとして仲間を持ちたい欲求はあるが、特定の「門」に帰属することに違和感がある。

📌 ホームとフィールドの喪失

室越氏が経験した、文化人類学的な「居場所」の変容。

  • 人類学者の3つのパターン:
    1. 「ホーム（自国）」と「フィールド（調査地）」が明確に分かれる人。
    2. どちらも「ホーム」になる人。
    3. どちらも「フィールド」になってしまう人（室越氏のケース）。
  • 室越氏の具体的エピソード:
    • キューバでの調査後、大分市役所に勤務。
    • キューバ社会は理解できた自負があったが、大分市役所では言葉（方言）も文化も分からず、解像度が著しく低下。
    • 「日本社会を分かっていたつもりで何も分かっていなかった」という感覚に陥り、ホームを喪失した。
  • 近代社会における可能性:
    • 肩書き（名乗り）と帰属意識（仲間意識）は分離可能。
    • 資本主義的な近代社会だからこそ、ハンドルネームのみで繋がるような「アノニマスな共同体」が成立しうる。
「肩書きという役柄を演じているうちに、その役柄自体が自分を作り変えていくのではないか」
