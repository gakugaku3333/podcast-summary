【土曜版 #83】深井龍之介さんと語る、資本主義と人文知、そして最近の日々

株式会社コテン代表の深井龍之介氏をゲストに迎え、新刊『人文知は武器になる』を切り口に、現代資本主義が抱える構造的欠陥と、それに向き合う経営者の苦悩、そして深井氏自身の極めて個人的な心境の変化を深く掘り下げた対談エピソードです。

人文知 資本主義の批判的考察 中長期合理性 国家の弱体化 株式会社の巨大化 燃え尽き症候群 内受容感覚 自己開示 知的好奇心 ルールチェンジ

📌 新刊『人文知は武器になる』の背景と反響

  • 執筆の動機：当初は本業への時間投資効率から執筆に消極的だったが、山口周氏との対談を機に、動画やポッドキャストでは届かない層へメッセージを届ける必要性を感じて出版を決意。
  • 驚異的な反響：発売2ヶ月で約6.7万部のベストセラー。本という媒体の「アーカイブ性」と「論理の精緻さ」を再認識。
  • 人文知への需要急増：ここ半年でビジネス界の人文知に対する関心が爆発。特にイスラエル・パレスチナ情勢などの地政学リスクが、既存のビジネスルールの限界を突きつけている。
  • 企業のインテリジェンス部門：30年〜100年単位の社会情勢を分析し経営に提言する部署の設置相談が急増（3年前の2社から現在は10社近くへ）。

📌 資本主義の「中心問題」と国家・企業の逆転構造

  • 国家の「老人化」と企業の「青年化」：
    • 国家：インフラ整備、教育、社会福祉などの公的機能を維持できなくなり、尊重はされるが力のない「老人」のような存在へ。
    • 巨大企業：国家レベルの資金、技術、人材を保有。生成AIや量子コンピュータ、軍事技術の主導権も民間が握る。
  • 資本主義の定義と特徴：深井氏は資本主義を否定せず、その特徴（利益へのインセンティブ、期待値の定量化など）を冷静に分析。
  • 最大の問題点「中長期合理性への合意不能」：
    • 資本主義社会では、短期的な利益（四半期決算など）には合意しやすいが、中長期的に合理的であっても、短期的に非合理なもの（環境問題、戦争抑止など）にリソースを割く合意が極めて困難。
    • 「寄付」や「ソーシャルインパクト投資」は、短期利益を上げた後の「間接的な投資」であり、効率が悪すぎる。

📌 経営者・深井龍之介の「現在地」と苦悩

  • 深刻な過労状態：身体的余裕がゼロ。朝起きた瞬間の第一声が「疲れた」。出張や講演の連続で心身ともに摩耗している。
  • 「はめ技」のダブルバインド：
    • やりたいことを進めたいが、進められないのもストレス。
    • 休まなければならないが、休んでいる間も仕事が進まないことがストレス。
    • この「どちらを選んでもストレス」という状況から抜け出せない。
  • 内受容感覚の低さ：自分の体の限界を察知する能力が低く、完全にダウンするまで「やりすぎ」に気づけない。
  • 「有名」であることへの嫌悪：本質的に有名になりたくない。期待値が上がりすぎることを恐れ、最近はあえて「性格の悪さ」や「不完全さ」を出すようにしている。

📌 価値観とパーソナリティの深層

項目 深井氏の視点・感覚
幸福の源泉 知的好奇心の充足。自分の案を凌駕する「完璧な代案」を出された時が最もハッピー。
行動原理 「愛情」よりも「非合理性への憤り（行き通りエンジン）」。なぜ戦争のメカニズムすら解明されていないのか、という問い。
生存欲求 長生きしたい欲求はほぼゼロ。後悔せずにスパッと死ぬのが理想。
公共性 元々は公共性が著しく低い「クズ」だと自認。だからこそ、他者のことを考える一般人の行動が純粋に不思議に見える。
自由意志 自分に自由意志はないと感じる。やらざるを得ない（埋め込まれた）使命感に従っている感覚。

📌 社会への提言と今後の展望

  • ルールチェンジの必要性：生成AIの登場や世界情勢の激変を受け、今この瞬間に株式会社のあり方を変えなければ、次世代に問題を先送りすることになる。
  • 「代案」の募集：単なる批判や否定ではなく、資本主義の中心問題を解決するための「建設的な代案」を求めている。
  • 実像の提示：マスメディア的な「清廉潔白なリーダー像」ではなく、不完全で性格の悪い部分も含めた「実像」を伝えることで、複雑な人間性を包摂できる社会を目指す。
  • 組織の代替不可能性：コテンという会社が深井氏個人の才能に依存しすぎている現状を課題視しつつ、それをどう組織化・代替していくかを模索中。
「中長期合理性に集中しないという選択をしているビジネスパーソンに問いたい。自分の子供の未来が安泰だと思っている人は少ないはずなのに、なぜ今、ルールを変えようとしないのか。僕は理想主義者ではなく、極めて冷徹に合理性の話をしている。」
