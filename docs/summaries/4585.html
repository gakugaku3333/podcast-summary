【情報の整理】加速主義：賛同せずとも無視できない「劇薬」の思想

Podcast「ちょいムズ本」第48回。ベンチャーキャピタリストの中路峻輔氏をゲストに迎え、木沢佐登志氏の著書『加速主義 増補新版』を題材に、資本主義とテクノロジーを限界まで加速させる思想の衝撃と、現代社会への予言性を読み解きます。

加速主義 (Accelerationism) ニック・ランド 新反動主義 (NRx) ベンチャーキャピタル 効果的加速主義 (e/acc) ピーター・ティール AGI (汎用人工知能) リバタリアニズム 民主主義の否定

📌 ゲスト・中路峻輔氏のスタンス

  • 俯瞰的・中立的な視点：起業家のように「社会をこうしたい」という強い主観的願望を持つより、社会の「重力」がどこにあるのかを観察するタイプ。
  • カウンター・オルタナティブ志向：メインストリーム（A）に対して、あえて別の選択肢（B）を探したくなる「天邪鬼」な性質を持つ。
  • 実験としての投資：自身の投資活動を「社会システムに対する実験」と捉え、仮説を世に問うプロセス自体に価値を置いている。

📌 紹介書籍：『加速主義 増補新版』とは

  • 著者：木沢佐登志（きざわ・さとし）。
  • 内容：ニック・ランドや新反動主義といった、既存の民主主義やリベラリズムを否定する過激な思想を紹介・解説した一冊。
  • 加速主義の定義：資本主義を抑制・制御するのではなく、むしろ限界まで加速させることで社会を崩壊・変容させ、その先に新しい未来を見出そうとする考え方。
  • 中路氏の衝撃：自身の生業である「ベンチャーキャピタル（VC）」の本質が、まさにこの「加速」にあると直感し、殴られたような衝撃を受けた。

加速主義の基本プロセス：制御を捨て、速度によってシステムを突破する

📌 現代における「加速主義」の具体化

  • 効果的加速主義 (e/acc)：
    • シリコンバレーのテック界隈でミーム化した動き。
    • AI（人工知能）の進化を倫理や規制で止めるのではなく、AGI（汎用人工知能）の完成まで一気に突き進むべきだとする主張。
    • 「AIは怖いから止めよう」という保守的な動きに対するカウンター。
  • VC（ベンチャーキャピタル）との親和性：
    • 非連続な成長（Jカーブ）を求めるVCの仕事は、本質的に社会の変化を加速させる装置である。
    • 現状維持を望む大企業中心の社会ではVCの出番はなく、変化が激しいほど相性が良い。

📌 政治・社会システムへの問い

概念 加速主義・新反動主義の視点
民主主義 合意形成に時間がかかり、テクノロジーの加速を阻害する「非効率なシステム」とみなす。
権威主義 トップダウンで意思決定できる体制（例：中国）の方が、技術加速には相性が良いという議論。
国家の企業化 国家も企業のように運営されるべきであり、不満があれば「声（Voice）」を上げるのではなく「退出（Exit）」を選択する仕組みを提唱。
リベラリズム 綺麗事や人間中心主義に縛られ、資本の本質的な運動を理解していないと批判。

📌 なぜ今、この本を読むべきなのか

  • 「無視できない」問い：全面的に賛同はできなくても、そこに書かれている論理を無視して現代のテクノロジーや資本主義を語ることはできない。
  • 2020年代の予言書：
    • かつてはトランプ政権に反対していたシリコンバレーが、規制を嫌ってトランプ支持に回るような現在の動きを予見していた。
    • AI、自動運転、バイオ技術など、もはや「止めることが不可能な変化」にどう向き合うかの指針となる。
  • 劇薬としての効能：
    • 「人間がどうなってもいいのか」という倫理的懸念はあるものの、既存の生ぬるい議論を打破する「劇薬」として機能する。
    • 特に金融やテック業界に携わる人間にとっては、自分の仕事の根源的な意味を問い直される経験になる。
「賛同はできないが、自分なりにこれに対する答えを出さなければならないと思わされる。2020年代の重要な考え方を提示している重要図書である。」
