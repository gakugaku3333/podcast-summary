深井・けんすうのまぼろし会議 35. これからの時代の価値観について

近代の基盤であった「個人の能力」による評価システムが崩壊しつつある現状を受け、これからの時代に何が新たな価値の源泉となるのかを議論。技術革新がもたらす社会構造の変化と、それに対する個人の生存戦略について深掘りします。

能力主義の終焉 港区女子の変容 ケアの価値 マスメディアの死 AIとボトルネック スモール・ストロング 教育と倫理 ホスト研究 メタ認知

📌 近代的な「能力ベース」の価値観の崩壊

  • 近代の物語の源泉： 努力、勉強、能力、成果、そしてその成果が個人のものになるという一連のサイクルがベースだった。
  • 崩壊の兆し： AIなどの技術革新により、個人の能力だけで成果を出すモデルが維持できなくなりつつある。
  • 新たな評価軸の仮説： 「能力」ではなく「何を支えているか（ケアしているか）」が評価の対象になるのではないか。

📌 価値観の転換事例：港区とグローバルリーダー

  • 港区界隈の変化： かつての「港区女子」的な属性よりも、現在は「子供の数」がステータスとして評価される傾向がある。
    • 少子化の中で、子供を産み育てていること、あるいはそれを支えていることが「偉い」とされる価値観へのシフト。
  • 富豪への視線の変化：
    • ジェフ・ベゾス： 自分のためにお金を使っているように見え、冷ややかな目で見られがち。
    • イーロン・マスク： 人類のために働いているという「大義」があるため、支持されやすい。
  • ケアの重要性： 自分のために稼ぐ人より、コミュニティや他者をケアする人の評価が高まる時代へ。

📌 ユニバーサルな評価の消失と分断

  • 共通指標の消失： 全人類が共通して「これが偉い」と思うユニバーサルな指標がなくなる。
  • エコーチャンバーの加速： インターネットにより、特定のコミュニティ内でのみ評価が高い状態が並列して存在するようになる。
  • マスメディアの終焉： テレビに出るメリットが激減。
    • 質の高い層はPodcastなどの深いメディアへ移動。
    • 有名になることは「叩かれるリスク」を増やすだけで、構造的なインセンティブが消滅している。

📌 技術革新が変える組織とボトルネック

項目 これまでの構造 これからの構造
組織の規模 数百〜数千人の大企業が強い 3人程度の少数精鋭がAIを駆使して巨大な力を保つ
ボトルネック 官僚的な事務作業、真面目な遂行 ケア、意思決定、倫理的判断
資本の価値 資金力があることが絶対的優位 限界費用がゼロに近づき、資本の価値が相対的に低下

📌 予測不能な時代への生存戦略

  • 教育の重要性： 社会がどう変わっても、江戸後期の教育が明治維新を支えたように、「今できる最大の教育」を施しておくことが唯一の備えになる。
  • 倫理のプリセット： 技術で何でも可能になるからこそ、「人殺しはダメ」「戦争はダメ」といった動かしてはいけない倫理を教育でセットしておく必要がある。
  • 「長時間労働＝偉い」の消滅：
    • 長時間労働は生産性の低さと多様性の欠如を招く。
    • 素晴らしい決断に時間は比例しない。むしろインプットを阻害する。

📌 深井氏の知的探求：未知の感覚を理解する

  • 研究対象： 自分がハマれないもの（ホスト、宗教、スポーツ、アイドル、お笑い）をあえて研究する。
    • ホスト： 歌舞伎町を「呪われた地」としつつ、案内人（守護神）と共にメタ認知の視点で調査。
    • スポーツ： 「負けて泣くなら最初からやらなければいいのに」という感覚の外側にある熱狂の構造を知りたい。
    • 海への恐怖： コントロールできない自然（海）が本能的に怖い。リゾートの海も「戦場」のように感じる。
  • 目的： 「なぜマジョリティがそれに熱狂するのか」を構造的に理解し、説明できるようになること。
  • メタ認知の限界： 自分の人生をメタ認知できていても、感情や行動を完全にコントロールできるわけではない（「自分もクソバカだな」と思って生きる）。
「技術革新があれば、社会規範は必ず壊れてきた。30年前のインターネット、20年前のスマホを経て、国家や株式会社の仕組みまでが『どんがら合称』される可能性が高い。」
