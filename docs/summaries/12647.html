【65-7】本気か茶番か？半信半疑な入学許可 〜招かれざる女子医学生の登校〜

エリザベス・ブラックウェルが医師を目指し、具体的な一歩を踏み出すエピソード。強力な支援者との出会い、精神的な危機と宗教的覚醒、そして28校からの拒絶を経て、奇跡的（かつ喜劇的）に医科大学への入学を果たすまでが描かれます。

ディクソン兄弟 医学書 解剖 精神的危機 神の啓示 ジェネバ医科大学 学生自治会 墓荒らし 道徳的覚醒

📌 医学への第一歩：強力なメンターとの出会い

  • ディクソン兄弟との出会い：元医師と現役医師の兄弟。エリザベスの志を応援し、個人指導や医学書の閲覧を許可した。
  • 一石二鳥の環境：兄弟の紹介で教職の仕事も得られ、資金を稼ぎながら医学を学ぶ理想的な環境を手に入れた。
  • 小さな成功体験：居候先の家政婦の頭痛を緩和させ、「ドクター・ブラックウェル」と呼ばれるようになる。
  • 禁忌への挑戦：
    • 昆虫の解剖：嫌悪感に耐えながら甲虫をナイフで切り、構造を学んだ。
    • 人体骨格模型：当時の女性としては「非道徳的」とされる行為だが、独学で知識を深めた。

📌 精神的絶望と「神の啓示」

  • 極限のメンタル：孤独、前例のない挑戦への不安、さらに寝床のノミの大量発生によるストレスで、一時は自殺を考えるほど追い詰められた。
  • 神秘体験：ある星明かりの夜、窓から見える山々に絶望を感じて神に叫んだ瞬間、圧倒的な光のような存在（栄光）が魂を満たした。
  • 覚悟の完了：この体験により、「自分の努力は人類の進歩という大いなる秩序にかなっている」という確信を得て、迷いが完全に消え去った。

📌 28校からの拒絶と社会の壁

著名な医師たちに願書を送るも、当時の価値観が厚い壁となって立ちはだかった。

周囲の反応・提案 エリザベスの信念
「看護婦（師）の方が向いている」という助言 正規の医師として学位を取ることに意味がある
「男性と一緒に解剖を学ぶのは反道徳的」 肉体の知識を得ることは恥ずべきことではない

直接訪問しても大声で笑われる、脅される 「諦めるくらいなら死んだ方がまし」と決意を固める

📌 ジェネバ医科大学：悪ノリが生んだ奇跡

29校目にして、ニューヨーク州のジェネバ医科大学から入学許可が届く。しかし、その内情は極めて不純なものだった。

📌 当時の医学教育の実態

  • 学生の質：田舎のやんちゃな若者が多く、近隣住民から苦情が出るほど。エリートは法律を学び、医者は「他にやることがない者」が選ぶ職業だった。
  • 遺体確保の闇：解剖用の遺体が不足しており、学生が墓荒らしをして調達することもあった（特に黒人や乳幼児の遺体が狙われた）。
  • 教育システム：
    • 2年間、全く同じ講義を繰り返すだけ。
    • 臨床実習（実際の患者を診る訓練）がほとんどない。
    • 卒業試験の基準が教授の裁量次第で、能力担保が不十分。

📌 初登校：静まり返る教室と世間の目

  • 教室の変容：普段は騒がしい男子学生たちが、エリザベスが入室した瞬間、麻痺したように静まり返った。彼女の真剣な眼差しが空気を変えた。
  • 街の反応：
    • 好奇の目：珍獣扱いされ、どこへ行っても視線が追いかけてくる。
    • 女性たちの拒絶：街の婦人方からは「はしたない」「悪い女か頭がおかしい」と冷笑された。
  • エリザベスの心境：
    • 強い自制心で周囲を無視し、自分を律した。
    • 同時に、注目されることで「自分が人生の主役になった」という高揚感も自覚し、それが新たなモチベーションとなった。
「私は目的を達成しなければならない。それを諦めるくらいなら死んだ方がましだ」
