ながらAIラジオ：2026年4月 AIニュースと新生活の展望

2026年4月3日収録。新生活シーズンにおけるAI業界の激動（Gemma 4登場、OpenAIによるメディア買収、Grokの自動翻訳など）を、AIコンサルタントのうすたく氏とメルカリAI推進担当の早川ゴミ氏が解説します。

📌 導入・ハードウェアと投資の考え方

  • 新生活とボーナスの使い道
    • うすたく氏はボーナスでMacBook Pro (M4 Max / 128GBメモリ / 8TBストレージ)を約100万円で購入。
    • 早川ゴミ氏はClaude Computer UseをMacで先行利用するためにMac miniを中古で購入（直後にWindows対応が発表されたが、常時稼働用として活用）。
  • Raspberry Pi 5の大幅値上げ
    • 16GBモデルが100ドル値上げ（2.5万円から4万円程度へ）。メモリやストレージの価格高騰が背景。
    • AI開発需要により、PCスペックの価格が今後さらに上昇する可能性を示唆。
  • 2025-2026年の重要性
    • 「数年後に決定版が出るまで待つ」のではなく、今このバブル期に時間を投下し、信頼と知識を蓄積することが将来の決定的な差になる。
    • 早川ゴミ氏は「2026年は会社員として得たお金を使い切る」勢いでAI・ロボットへの自己投資を継続中。

📌 Google「Gemma 4」とローカルLLMの台頭

  • Gemma 4の登場
    • Googleのオープンモデル。ベンチマークで米国製LLMとして1位を獲得（中国勢のQwen等も強力）。
  • ローカルLLMのメリット
    • オフライン動作: 電波のない環境（飛行機など）でも利用可能。
    • プライバシー・コスト: 自分のPCで動かすためAPI料金がかからず、機密情報も扱いやすい。
    • リスクヘッジ: 特定のプラットフォーム（Anthropic等）が停止しても業務を継続できる。
  • 推奨ツールと環境
    • PC用: LLM Studio, Ollama。
    • スマホ用: Localy AI（iPhone 16 Pro Max等でQwenやGemmaが動作）。
    • 推奨スペック: メモリ64GB以上あると快適。うすたく氏の128GB環境ではサクサク動作。

📌 OpenAIによるメディア「TVPN」買収

  • 買収の概要
    • シリコンバレーのニュースをスポーツ実況のように熱狂的に伝えるポッドキャストメディアTVPNをOpenAIが買収。
    • 買収額は数億ドル（数百億円規模）と推定。
  • 買収の背景と意義
    • AI時代において、AIで代替できない「人間の個性」「信頼」「ライブ感」に価値が置かれている。
    • 創業1.5年、毎日3時間の生配信を継続した結果の評価。
    • 「ながらAIラジオ」も生配信の導入や、属人的な信頼獲得の重要性を再認識。
