はやぶさ2の新たな挑戦：小惑星「鳥船」への接近と未来の火星探査

2026年7月7日の七夕に合わせ、JAXAの探査機「はやぶさ2」が小惑星「鳥船（トリフネ）」へのフライバイ（接近観測）に成功したニュースを中心に、日本の宇宙探査の現在地と、次なる火星衛星探査計画（MMX）について解説しています。

はやぶさ2 JAXA 鳥船 (2001 CC21) フライバイ 地球防衛 (プラネタリー・ディフェンス) MMX 火星衛星探査 サンプルリターン

📌 はやぶさ2：小惑星「鳥船」へのフライバイ成功

  • ミッションの概要：地球から約1億km離れた小惑星「鳥船（トリフネ）」に接近しながら観測する「フライバイ」に成功。
  • ターゲット「鳥船」：
    • 正式名称は2001 CC21。
    • 日本神話に登場する神の乗り物にちなみ、公募によって命名された。
    • 直径わずか約500メートルの極めて小さな天体。
  • はやぶさ2の現状：2020年に小惑星リュウグウのサンプルリターンを終えた後、機体は健全な状態を保ち、拡張ミッションとして旅を継続中。

📌 驚異的な技術難易度：「北海道から沖縄の1円玉を射抜く」

今回のミッションは、あらかじめ「極めて困難」とされていました。その理由は以下の通りです。

  • 超高速での通過：はやぶさ2は時速約18,000kmという猛スピードで移動している。
  • カメラの制約：カメラが固定されているため、接近の瞬間にレンズが正確に鳥船を向くよう、事前に軌道と姿勢を完璧に計算する必要があった。
  • 自律判断の必要性：距離が遠すぎて地球からのリアルタイム通信が間に合わないため、探査機自身が自律的に判断して行動しなければならない。
  • 比喩表現：JAXAはこの難易度を「北海道にある1円玉を、沖縄から弓矢で射抜くようなもの」と表現している。

📌 地球防衛（プラネタリー・ディフェンス）への貢献

  • 技術実証の意義：地球に衝突する恐れのある小惑星を調査し、その軌道を変えるための基礎データを得る「地球防衛」の側面を持つ。
  • 世界第2位の快挙：小惑星の軌道変更に関する技術実証を行ったのは、アメリカのNASAに続き、日本が世界で2番目となる。
  • 信頼性の証明：2014年の打ち上げから長期間、大きな故障なく運用され続けていることは、日本の探査機技術の極めて高い信頼性を裏付けている。

📌 次なる主役：火星衛星探査計画「MMX」

はやぶさ2の技術を継承し、いよいよ火星圏への挑戦が始まります。

  • MMX (Martian Moons eXploration)：火星の衛星「ホボス」に着陸し、サンプルを持ち帰る計画。
  • 世界初の挑戦：火星圏への探査は過去に事例があるが、すべて片道。「火星圏からの帰還（サンプルリターン）」に成功すれば世界初の快挙となる。
  • 3つの大きな研究テーマ：
    • 形成の謎：火星とその衛星がどのようにして生まれたのか。
    • 生命の可能性：かつての火星環境における生命誕生の可能性の探求。
    • 人類の居住：将来的な人類の火星居住に向けた基礎データの収集。

📌 補足：七夕の豆知識

  • 時期のズレ：本来の七夕は現在の8月頃（旧暦）に行われていたが、明治時代の改暦により梅雨時期の7月にずれてしまった。そのため雨が降ることが多い。
  • 短冊の意味：願いを文字に書くことで、その願いが意識化され、行動に影響を与えることで実現に近づくという側面がある。
「北海道の一円玉を沖縄から射抜く」ほどの精密な技術が、今や地球を守り、火星という新たなフロンティアを切り拓こうとしています。
