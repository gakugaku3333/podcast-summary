#006 展示会では絶対に見せない！？ヒューマノイドがまだできないこと

展示会やプロモーション動画では完璧に見えるヒューマノイドロボットの「不都合な真実」を深掘りするエピソード。自律動作の限界、遠隔操作の裏側、ハードウェアとしての脆弱性など、実用化に向けたリアルな課題が語られています。

📌 「自律」の裏側にある遠隔操作の現実

  • 1X Technologies（ノルウェー）の事例
    • 家庭用ロボット「NEO」を開発する同社のCEOベルント氏は、「未知の家事は人間が遠隔操作でやっている」と公言している。
    • 100%自律で動いているように見えても、実際には人間が教えてあげたり、学習のためのデータを収集したりするプロセスが不可欠。
  • 展示会での「ラジコン」状態
    • 多くの企業が自律動作を謳いながら、ブースの裏側でエンジニアがコントローラーを操作しているケースが多々ある。
    • 安全性の配慮や、LLM（大規模言語モデル）による不確実な挙動を避けるために、現状では遠隔操作が主流となっている。
  • テスラ「Optimus」のハプニング
    • 2023年12月のデモ中、転倒した際に「顔面に手を置いてゴーグルを外すような動作」を見せた。
    • この人間特有の仕草により、裏でVRゴーグルをつけたオペレーターが操作していたことが露呈してしまった。
  • 誠実な情報公開の重要性
    • 川崎重工などは、テレビ番組で「遠隔操作であること」を明示した上でデモを行っており、技術の現在地を正しく伝える姿勢が評価されている。

📌 遠隔操作技術のポジティブな側面

  • プロフェッショナルの拡張
    • 1人の熟練者がオフィスにいながら、世界中にある10箇所の現場のトラブルをヒューマノイド経由で解決できる。
  • 社会参画の新しい形
    • 引きこもりの人や外出が困難な人が、ロボットの体を使って労働・社会参加できる可能性がある。
  • 医療現場での実績
    • 遠隔操作ロボットは医療分野で先行しており、高度な技術を持つ医師が遠隔地から手術を行うなどの応用が期待されている。

📌 技術的な高い壁：歩行よりも難しい「掴む」動作

  • マニピュレーション（把持）の困難さ
    • 二足歩行よりも、手で物を掴むことの方が圧倒的に難易度が高い。
    • 理由1： 人間の手は40億年かけて進化した精密機械であり、10年程度の開発では追いつけない。
    • 理由2： 5本指・3関節にすると、15箇所の可動部が故障リスクになる。そのため、実用的には3本指や2本指のピッキングが選ばれることが多い。
    • 理由3： 触覚センサが手のひら全体をカバーできておらず、握り直しの微調整が困難。
  • 制御手法のジレンマ
    • 教科学習： 滑らかな歩行は得意だが、目標地点との間に数十センチのズレが生じることがある。
    • 古典的制御： 数ミリ単位の精密な動作が必要な工場などでは、あえて古い制御手法が選ばれる。

📌 運用コストと物理的な限界

  • 莫大なエネルギー消費
    • ヒューマノイドは立っているだけで、バランスを保つために常に計算とモーター駆動を行っている。
    • 電源を切ると自重で倒れてしまうため、待機中も電力を消費し続ける。
  • 「1人1台」では足りない現実
    • バッテリー持ちや故障リスクを考えると、1人分の仕事をさせるには2〜3台をシフト制で回す必要がある。
  • 高額な修理費用
    • Unitree G1の事例では、1回の転倒で本体価格の約30%の修理コストがかかるという試算もある。
    • 転倒するたびに数十万〜数百万円のコストが発生するリスクがある。

📊 ヒューマノイド導入の理想と現実

項目 展示会・プロモーションでの見せ方 現場でのリアルな実態
動作 100%自律でスムーズに動く 裏で人間が遠隔操作（テレオペ）
精度 どんな物でも器用に掴む 数センチのズレで失敗、触覚が未熟
稼働時間 人間のように長時間働く バッテリー消費が激しく、複数台必要
安全性 人間と安全に共存 電源断で60kgの塊が倒れてくるリスク

📌 今後の展望：安全評価とインフラ整備

  • 安全規格の策定
    • ヒューマノイドが急に倒れた際の人体への影響など、安全基準は現在策定中。
  • ロボット保険の必要性
    • 自動車保険のように、ロボット自体の破損や、ロボットが周囲に与えた損害をカバーする専用保険の発達が予測される。
  • 導入時の見極めポイント
    • 「自律で動けます」と言い切る業者は怪しい。
    • 「遠隔操作が必要」「2〜3台での運用が必要」といったリスクと現実を提示できるベンダーが信頼できる。
