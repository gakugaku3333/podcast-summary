『資本主義と、生きていく。』発売後の反響とメディア行脚の舞台裏

著者である品川氏が、著書発売から4ヶ月間で行ってきた膨大なメディア出演の意図や、その過程で得た気づき、そして思想家・東浩紀氏との対談で感じた緊張感と「対話」の本質について語る雑談エピソードです。

資本主義と、生きていく。 品川哲志 樋口聖典 東浩紀 石田健 超相対性理論 PIVOT J-WAVE 別所哲也 荒木博行 渡邉康太郎 母親アップデートコミュニティ 創発 西田幾多郎

📌 メディア露出の戦略と目的

  • ゲリラ戦としての宣伝活動
    • 「本は手売りではないが、フェイス・トゥ・フェイスで喋って買ってもらうことが大事」という信念。
    • 編集者の出口氏と共に「呼ばれたらどこへでも行く」というスタンスで、ポッドキャスト、Webメディア、ラジオ等へ積極的に出演。
  • ポッドキャストへの恩返し
    • 自身の番組（日本一たのしい哲学ラジオ）があったからこそ出版に辿り着いたという感謝。
    • ポッドキャスト界隈からの依頼は基本すべて受け、既に30〜40件ほどのゲスト出演をこなしている。
  • 「開かれた対話」の場としての活用
    • 本の内容そのものの解説だけでなく、本をきっかけにした「家族」「キャリア」「女性の働き方」など、本では書ききれなかったテーマについて対話する場としている。

📌 樋口氏から見た「相方の活躍」

  • 「推し活」に近い感情
    • 自分自身がメディアに出る時はフラットだが、相方である品川氏が有名番組に出ているのを見ると、家族のような、あるいは「推し」を応援するような不思議な喜びを感じる。
  • 品川氏の成長への期待
    • 場数を踏むことで、相手との関係性や尺に合わせた「切り口」の整理が洗練されていく過程を、ミュージシャンのライブツアーになぞらえて評価。

📌 「対話」と「議論」の違い

  • 創発（そうはつ）の重視
    • 対話のラリーを通じて、一人では思いつかなかった新しい視点が生まれるプロセスに価値を置いている。
    • メディア出演においても、単なる情報の伝達ではなく、対談相手と「また話したい」と思える関係性を築くことを重視。
  • 出演後の反省
    • 「もっとこう言えばよかった」「自分の役割はこれでよかったか」という悔しさを、成長のための「試合」の経験として捉えている。

📌 今後の展望と決意

  • 「本を残すこと」が人生の目的
    • 今回の出版と反響を通じて、改めて「自分の思想を本という形で世に残すこと」が最大の幸せであると再認識。
    • 2026年〜2027年にかけて、さらに新しい本を作ることに人生のリソースを割いていく決意。
