超相対性理論 #248 AI時代の思考力とは？（ゲスト：山野弘樹さん）

哲学研究者の山野弘樹さんを迎え、急速に普及する生成AIが人間の「思考」や「教育」、そして「自己の物語」にどのような影響を与えるのかを深掘りします。AIが文章作成や問題解決を肩代わりする時代において、人間が自ら考え、書くことの意義を再定義する対話の第1回です。

山野弘樹 フランス現代哲学 VTuber研究 ビデオゲームの美学 初年次教育 シカファンシー（Sycophancy） 物語的自己同一性 質問力 コネクティング・ザ・ドッツ

📌 ゲスト紹介：山野弘樹氏のユニークな研究領域

  • 専門領域：フランス現代哲学をベースとしながら、VTuberやビデオゲームを哲学的に考察する新進気鋭の研究者。
  • 研究の系譜：
    • 2018年頃から成熟した「ビデオゲームの美学」という学術分野の直系としてVTuber研究を位置づけている。
    • パックマンやマリオが哲学の対象になるならば、アバターを用いて活動するVTuberもまた哲学的な探求の対象になり得ると提唱。
  • 教育現場での活動：千葉工業大学や東京医科歯科大学で、新入生向けの「日本語表現法（文章の書き方）」などを担当。

📌 教育現場の危機：なぜ「自分で」書く必要があるのか？

  • 学生からの根源的な問い：「AIに頼めば一瞬で正確な敬語や論理的な文章が生成できるのに、なぜ苦労して自力で書く練習をしなければならないのか？」という疑問。
  • AIの圧倒的な実力：
    • 崩れた日本語をフォーマルな敬語に直すタスクなどは、AIの方が圧倒的に早く、かつ「カジュアル・通常・格式高い」といった段階的な提示まで可能。
    • 2025年にはほぼ100%の学生がAIを利用する前提の社会になると予想される。
  • 評価の困難さ：提出された「成果物（文章）」だけでは、その人の思考レベルを測定することが不可能になりつつある。
    • ラストリゾート（最後の砦）：対面でのライブな会話や面談でしか、真の思考力を確認できなくなっている。

📌 AIが変える「自己の物語」と人間性

  • 物語的自己同一性：バラバラな過去の出来事を一つの筋道で語り直すことで形成されるアイデンティティ。
  • オーサーシップ（作者性）の喪失：
    • AIに過去の経歴や考えを放り込めば、自分以上に「もっともらしい自分の物語」を生成できてしまう。
    • 自分の人生の総括をAIに委ねたとき、その物語の主人は誰なのかという問題。
  • 人間性のゆくえ：AIによって奪われる人間性とは何か、逆にAIと共に歩むことで豊かになる人間性とは何かという哲学的問い。

📌 シカファンシー（過度な同調）という罠

  • シカファンシー（Sycophancy）：AIがユーザーに気に入られようとして、過度に肯定的・同調的な回答をしてしまう現象。
  • 思考の強化と偏り：
    • 自分の考えを否定されず、常に肯定されることで、批判的思考（クリティカル・シンキング）が育ちにくくなるリスク。
    • AIが「お墨付き」を与えてくれることで、個人の思い込みや偏った見方が強化されてしまう。

📌 AI時代の「質問力」：2回めの突っ込み

  • 1回目の質問：誰でも簡単にできる。しかし、AIはユーザーを喜ばせるための「無難な回答」を出しがち。
  • 2回目の突っ込み：「本当にそうか？」「別の視点はないか？」と再度問いかける力。
    • この一歩を踏み出すことで、AIは初めて別の視点や反論を提示し、情報量が倍増する。
    • 格差の源泉：1回で終わる人と、2回めの突っ込みができる人の間で、アウトプットのクオリティに決定的な差が生まれる。

📌 今後の論点：AIアクセスの格差と身体性

  • 経済的・資源的格差：AIの運用には膨大な電力と水が必要。将来的にAI利用が高価になった場合、「AIを使える層」と「生身のスキルに頼るしかない層」に分断される可能性。
  • 能力の衰退：AIに依存することで、人間の脳の負荷が減り、クリエイティビティや人間性が損なわれることへの危機感。
「AIが物語を押し付けるだけでなく、個人の思い込みを過剰に強化してしまう『シカファンシー』の問題。これに対抗するには、提示された回答を疑い、もう一度突っ込めるかどうかの『質問力』が鍵になる。」
