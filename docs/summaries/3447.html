📌 ポッドキャストアワードの振り返りと今後の展望

  • 一般部門上位10作品に選出：開始1年目でTravis Japanなどと並びランクイン。
    • 結果は入賞を逃し、優勝は「ゆる言語学ラジオ」。
    • 専門性の高い番組よりも、雑談系・エンタメ系が強い傾向を再認識。
  • 番組のクオリティ向上への決意：リスナーが人に勧めたくなるような内容を目指す。
  • スポンサー募集のブレスト：
    • AI系（人材・転職）以外の、体験・教養・文化系のスポンサーを希望。
    • 例：金沢21世紀美術館、科学系、レンタカーなど。

📌 Manus Desktop版の登場とツール戦略

  • デスクトップ版の特徴：
    • ブラウザ版と異なり、ローカルフォルダへのアクセス権限を付与可能。
    • 領収書ファイルの整理やファイル名の自動変更などが容易に。
    • 独自のデータベース機能やデプロイ機能（Manus.space）が強力。
  • ツールの使い分けに関する持論：
    • 「どのツールが良いか」で悩む時間は無駄。
    • 1つのツールを極めることが重要。明確にできないことが出てくるまで使い倒す。
    • うすたく氏はManus派、早川ごみ氏はClaude派と好みが分かれる。

📌 常時稼働PCとハードウェアの重要性

  • AIエージェントのための常時稼働：
    • 寝ている間もAI（Claude CodeやCore Arc）を動かし続けるニーズが増加。
    • ノートPCを閉じた状態での稼働は排熱問題で画面が焼けるリスクがある（早川氏の体験談：修理費5万円）。
  • 推奨ハードウェア：
    • Mac miniやMac Studioが合理的。
    • ローカルLLMを快適に回すには、メモリ・GPUを積んだ高スペック機（80万円〜）が必要になることも。
  • 税制への不満：100万円程度のPCも一括経費（減価償却ではなく）で落とせるようにしてほしいという主張。

📌 GensparkのLINE対応とインターフェース競争

  • LINE連携の意義：
    • SlackやDiscordよりも、一般ユーザーにとって馴染みのあるLINEでAIを呼び出せる。
    • 日本市場を意識した戦略的な動き。
  • 「面（インターフェース）」の取り合い：
    • AIの性能差が縮まる中、どこで使えるか（PC、スマホ、LINE、Officeツール内）という接点の奪い合いが激化している。

📌 NotionとMicrosoft Copilotの現状

  • Notionのアップデート：
    • 見出し4 (H4) が追加。地味だが構造化に便利。
    • カスタムエージェントが2026年5月4日まで無料で試用可能。
    • 組織でのナレッジ共有には、個別のAIツールよりNotion AIの方が引き継ぎや参照性が高い。
  • Microsoft Copilotへの不満：
    • ExcelのWeb版でしか動かない、挙動が不安定など、本家ツールとしての使い勝手に課題。
    • 外部AI（Claude等）の方がOfficeファイルを扱いやすいという逆転現象。

📌 Claude Core Arcによる「AIネイティブ」な働き方

  • Core Arcの利点：
    • エンジニアでなくても使いやすいUI。
    • Dispatch機能：外出先からスマホで、自宅のPCで動くCore Arcに指示が出せる。
  • AIに歩み寄るツール選び：
    • AIが操作しやすいツール（Notion, PowerPoint, Canva等）に業務を寄せる。
    • FigmaやGoogleスライドは、現状AIからの操作やAPI連携に制限が多い。
  • 具体的な活用例：
    • Notionに放り込んだ雑多なメモを、Core Arcが毎日1回整理してアーカイブする。
    • 確定申告用に、ドライブ内のカード明細を自動集計させる。

📌 未来予測：ユニバーサル・ハイ・インカム (UHI)

  • ベーシックインカムの先：
    • イーロン・マスク提唱。ロボットが全労働を代替し、人間が働かなくても「高い水準の生活」ができるお金が配られる世界。
  • 「働く」ことの価値変容：
    • 10年後には「昔の人はお金のために皿洗いや事務作業をしていたのか」と驚かれる可能性がある。
    • 労働から解放された人間は、コミュニティ、教育、地産地消、趣味（クッキーを焼く等）に生きがいを見出すようになる。
  • 生活の価値：
    • AI界隈のインフルエンサーこそ、効率化だけでなく「生活感（食事、料理、趣味）」を大切にすべき。
