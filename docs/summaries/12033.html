📌 「覚書き（メモランダム）」の言葉の背景

  • 和語としての由来：外交用語の多くが漢語由来である中、「覚書き」は江戸時代から続く日本古来の和語。
  • 概念の継承：明治時代に西洋から「備忘録」「共通の取決め」という概念が入ってきた際、新造語を作らずに既存の「覚書き」を当てはめた。
  • 現代の意義：21世紀の国家間の約束事を表す言葉として、江戸時代からの言葉が生き続けている。

📌 米イラン戦闘終結とG7の対応

  • 事前通告の欠如：アメリカとイスラエルによるイラン攻撃は、イギリス以外のG7諸国に事前通告がなかった。
    • イギリスは基地使用を打診されたが拒否。
    • G7内でアメリカと他国の立場の違いが表面化している。
  • 戦闘終結に向けた覚書き：
    • 署名者：アメリカ側はトランプ大統領とバンス副大統領、イラン側はガリバフ国会議長。
    • 署名方式：デジタル署名が完了しており、6月19日にスイスで正式署名予定。
    • 主な内容：ホルムズ海峡の全面開放（19日〜）、制裁緩和の枠組み。
  • サミットの論点：軍事的対立から外交的解決へ舵を切る中、中東の今後やエネルギー安全保障の再構築が議論される。

📌 日本が提案する「エネルギー安全保障3原則」

高市首相がサミットで提案する、日本の主導権確保を狙った共通認識の形成案。

原則 内容の詳細
1. 安定供給の確保 エネルギー資源の途絶を防ぐための国際的な協力体制。
2. サプライチェーンの強靭化 特定の供給源に依存しない多様なルートの構築。
3. 移行期の安全性担保 クリーンエネルギーへの移行過程におけるエネルギー不足やリスクの回避。

📌 経済安全保障：重要鉱物を巡る対立と日本の役割

  • アメリカの独自計画：
    • 重要鉱物（クリティカルミネラル）の価格設定を、国防総省のAIモデルに基づき決定するシステムを提唱。
    • 目的は中国を睨んだ価格統制と維持。
  • 各国の反応：
    • G7他国は懐疑的。特にヨーロッパ諸国はアメリカへの権力集中を警戒。
    • アメリカ国内の産業界でも賛否が分かれている。
  • 日本の橋渡し役：
    • G7各国での鉱物資源の共同備蓄制度の立ち上げを支援・提案。
    • アメリカの強硬姿勢とヨーロッパの警戒感の間を繋ぐ存在として注目されている。

📌 G7の変容と「G9」への拡大構想

シンクタンクCSISの分析に基づく、G7の実効性と枠組みの見直し議論。

  • 影響力の低下：1980年には世界GDPの60%を占めていたが、現在は44%まで減少。人口比も10%を割り込んでいる。
  • 課題の広域化：AI、気候変動、食料安全保障、対中国など、扱うテーマは拡大し続けている。
  • 新メンバーの台頭：
    • 韓国：デジタル科学技術分野でアメリカに次ぐ世界2位の評価。
    • オーストラリア：多くの指標でG7諸国に匹敵する成績。
  • 実効性の向上：CSISは、多極化する世界に対応するため、韓国・オーストラリアを加えたG9への拡大と制度の見直しを提言している。
世界のリーダーが集まり議論すること自体の重要性は揺るがないが、形骸化を防ぎ、具体的なルール形成ができる場へとアップデートできるかが問われている。
