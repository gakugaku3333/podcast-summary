【土曜版 #70】イラン危機・円安・AI競争。歴史から読み解く、日本の進むべき道

ジャーナリストの船橋洋一氏をゲストに迎え、新著「戦後敗戦」を軸に、日本が直面するエネルギー危機、半導体・AI競争、そして国家としての再建策について深く掘り下げます。過去の成功と挫折から、現代の難局を乗り越えるための教訓を整理します。

📌 概念：なぜ今「戦後敗戦」なのか

  • 「戦後敗戦」の定義： 第二次世界大戦後の80年を40年ずつに分けた際、後半の40年（平成以降）で日本が経験したじわじわとした衰退と挫折。
  • 象徴的な数字： 1988年にOECD諸国で2位だった一人当たりGDPが、2024年には24位まで転落。
  • 現状認識の必要性： 「茹でガエル」状態を脱するため、一度「負けた」と割り切り、覚悟を決めて出直すことが不可欠。
  • 日本人の特性： 「自利品（じりひん）」で耐えてしまう傾向があるが、このままでは「土下品（どげひん）」に陥る危惧がある。

📌 エネルギー危機：イラン情勢とホルムズ海峡の封鎖リスク

  • 事実上の封鎖： 物理的な封鎖がなくとも、リスクの高まりにより船主が航行を控えることで、経済的には封鎖と同じ状態に陥る。
  • イランの体制： 革命防衛隊は単なる軍事組織ではなく、経済・生活に根ざした「生活共同体」であり、指導者の除去だけで崩壊するほど単純ではない。
  • 米国の変化： シェール革命によりエネルギー自給を達成した米国は、中東への関与を弱める（ピボット）動機がある。将来的に「シーレーン維持のコストをお前ら（日本等）が払え」と言い出すリスク。

📌 半導体・ネット敗戦の教訓

  • 半導体： 1990年代初頭、日本はメモリで世界一だったが、米国の圧力（数量目標の受け入れ）と円高により価格決定権を喪失。
  • 構造的失敗： 垂直統合型の「総合電機メーカー」に固執し、ファブレス・ファウンドリの分業化やPC・スマホ向けロジック半導体の波に乗り遅れた。
  • ネット・AI： 英語圏の「ハイパースケール（15億人の市場）」に対し、日本はスケールメリットで勝負できなかった。
  • リバースエンジニアリングの必要性： 米国がイラン製ドローンを解析して自国生産するように、日本も「後発組」としての自覚を持ち、貪欲に学ぶ姿勢が必要。

📌 未来への提言：目指すべき2つの国家像

1. 国民安全保障国家

  • 有事への備え： 福島原発事故やコロナ禍で露呈した「脆さ」を克服する。
  • 新・社会契約： 有事における政府の権限と国民の義務を明確化する。空気や忖度で動くのではなく、法的根拠に基づいた対応が必要。
  • サイバー・AI防衛： 個人のプライバシーや情報がリアルタイムで把握される時代、国民も当事者として守り・守られる関係を築く。

2. 企業家国家

  • 政府の役割： 「小さな政府」が最善というネオリベラル的発想を捨て、政府が戦略的に需要を創出し、技術の社会実装を後押しする。
  • アライド・スケール（同盟国スケール）： 日本単独ではなく、米国や同志国と連携して中国の巨大スケールに対抗する。
  • 尖った人材の活用： 明治時代の「お雇い外国人」のように、閣僚より高い報酬を払ってでも外部の高度人材や多様な人材を登用する。

📌 結論：変革のトリガー

  • 政治のリーダーシップ： 縦割りや個別最適を打破し、全体のエゴ（ビジョン）を描ける政治家が必要。
  • 有権者の目： 単なる「分配」のおねだりではなく、新しい国の形をデザインする意志と野心を持った政治家を見極め、夢を託す。
  • 後発の自覚： かつての「経済大国」という幻想を捨て、明治維新や戦後直後のように「後発組」としてキャッチアップする覚悟を持つこと。
