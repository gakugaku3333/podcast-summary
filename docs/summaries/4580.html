AI時代は「倫理」が不可欠。土地に根ざした学びをいま行う意味

龍谷大学副学長の深尾昌峰氏をゲストに迎え、2027年に新設される「環境サスティナビリティ学部」と「情報学部」の狙い、そしてAIや資本主義の揺り戻しが起きている現代社会における「倫理」と「地域に根ざした教育」の重要性について深掘りします。

龍谷大学 深尾昌峰 環境サスティナビリティ学部 情報学部 インパクト投資 ネイチャーポジティブ経営 環境DNA 自省利他 コレクティブインパクト ローカルGX ビワコオーツキャンパス

📌 ゲスト：深尾 昌峰（ふかお まさたか）氏の経歴

  • 社会起業家と研究者の両輪： 1998年に京都NPOセンター事務局長に就任。2012年には「プラスソーシャル」を起業。
  • 龍谷大学での歩み： 2010年に法学部准教授、その後教授を経て、2022年より副学長を務める。
  • 専門領域： インパクト投資のフレームワーク構築など、経済合理性と社会的価値を統合する活動に従事。

📌 現代社会の潮流：行き過ぎた資本主義の揺り戻し

  • 「揺らぎ」の時代： これまでの当たり前が崩れ、人間性の回復や持続可能な社会への問い直しが起きている。
  • インパクト投資の普及： 投資家が「儲け」だけでなく、人権や環境を重視し始めた。経済合理性だけでは地球が持たないという危機感の表れ。
  • ポスト・トランプ時代の日本： アメリカが環境政策から一歩引く可能性があっても、脱炭素の流れは科学的な規定路線。日本がこの分野でリードする大きなチャンスと捉えるべき。

📌 2027年新設：2つの学部の狙いと特徴

龍谷大学創立400周年（2039年）に向けたビジョン「構想400」の一環として、滋賀県大津市の瀬田キャンパスに設置される。

📌 学問の統合と「現場体験」の重要性

  • 縦割りからの脱却： 自然科学（環境）と社会科学（経営）を統合的に学ぶ。専門性を深めるだけでなく、領域を往復して価値を生む「イノベーションの通訳者」を育てる。
  • 身体性を伴う学び： 知識だけでなく、実際に森や琵琶湖などの現場に行き、五感で自然を感じる「原体験」を重視。これが行動変容の源泉となる。
  • 実務家教員の登用： 企業の第一線で悩む実務家を招き、建前ではない「現実の課題」を学生と共に考える場を作る。

📌 AI時代における「仏教・倫理」の役割

  • 「何でもできる」時代の羅針盤： テクノロジーが進歩し、物理的制約がなくなる中で、「何をすべきか」という倫理観（背骨）が問われている。
  • 自省利他（じせいりた）： 龍谷大学の見学の精神。自分を省み、他者の幸せを願う精神が、AIを「人を傷つける道具」ではなく「幸せにする道具」として使うための鍵となる。
  • 経営者の関心： 変化の激しい時代において、過去の成功体験が通用しない。そのため、判断の基盤となる哲学や仏教を学びたいと考える経営者が増えている。

📌 キャンパスの再定義：ビワコオーツキャンパスへ

  • コレクティブインパクト： 大学が閉じた存在ではなく、多様なセクター（企業・行政・市民）が混じり合い、社会変革を起こす拠点となる。
  • 偶発性の価値： オンラインで知識は得られるが、キャンパスでの「たまたまの出会い」や「ブレストでの化学反応」は代替できない。
  • 大学の「極寒の時代」への対抗： 18歳人口が激減する中、地域にとって「なくてはならない存在」になれるか。土地に根ざし、地域のニーズと研究をマッチングさせることが生き残りの道。
「大学があることで、地域が良くなった、誰かが喜んでくれた。その小さな積み上げこそが、400周年を迎える大学の信頼の礎になる。」
