📌 Claude 4.8 Opus の登場とモデル戦略

  • 驚異的なリリース速度：前回の4.7からわずか40日で新モデルが登場。
  • モデルラインナップの現状：
    モデル名 バージョン 特徴
    Opus 4.8 最上位モデル。今回アップデート。
    Sonnet 4.6 速度と性能のバランス。
    Haiku 4.5 高速・軽量モデル。
  • 性能の実感：
    • ウスタクト氏によるタワーディフェンスゲームの制作テストでは、断トツの賢さを発揮。
    • 「モデルを一度弱体化させてから、元の性能で出し直してユーザーを喜ばせる」という海外のミーム（戦略説）についても言及。

📌 Claude Code の新機能：Ultra Code Mode & Workflows

  • Ultra Code Mode：
    • コマンド /effort で設定可能。従来のMaxを超える最高設定。
    • 起動するとギラギラした紫色のエフェクトが表示される遊び心のあるUI。
  • Workflows（ワークフロー）機能：
    • チームエージェントに近い挙動。全体の設計からエージェントの割り当てまでを自動化。
    • /workflow コマンドで、現在の進捗（プランニング、実装、統合、テスト、レビュー）を可視化。

📌 次世代モデル「Mytos (ミュトス)」の噂

  • 数週間以内に登場か：Anthropicのリリースノートに示唆。
  • サイバーセキュリティ特化：
    • 現在は軍事・金融などの特定パートナーのみに公開されている。
    • 脆弱性発見能力が極めて高く、「守れるということは攻められる」というリスクも孕む。
    • ジェイルブレイク（制限解除）により、悪用される懸念についても議論。

📌 NotebookLM の同期機能アップデート

  • Googleドライブ連携：Google Docsなどの更新が自動的にNotebookLMへ反映される。
  • 手軽なRAG（検索拡張生成）：
    • 専門知識がなくても、社内マニュアルや資料を読み込ませたチャットボットが簡単に作成可能。
    • 大学法人（近畿大学など）での導入事例も増加。
  • ソース選択の柔軟性：特定のソースを選んで回答させる機能が向上。

📌 AI時代の「偏愛」と個人開発トレンド

  • 偏愛（こだわり）の重要性：
    • AIで誰でも「それなりのもの」が作れる時代だからこそ、個人の狂気的なこだわりが差別化になる。
    • ウスタクト氏のこだわり：「現金のみの店には絶対行かない」などの生活ルール。
    • 早川ゴミ氏のこだわり：「ものづくり」への執着。
  • 話題の個人開発「国会議員マップ」：
    • 地方の経営者が開発。議員の発言や税金の使われ方を可視化。
    • X（旧Twitter）で13万人フォロワーを獲得する爆発的なグロースを記録。
  • AI入力のトレンドと誤解：
    • 「ローマ字入力が効率的」というバズり記事への反論。実際は出力精度が落ちるため、普通に日本語で会話するのが最善。
    • 「原始人プロンプト」：テキスト量を減らし、本質的なやり取りに絞るテクニック（例：「俺コード実装、お前レビュー」）。
