なぜ日本からパワハラがなくならないのか：文化的・構造的要因の解明

神奈川県立保健福祉大学大学院の津野香奈美教授をゲストに迎え、日本社会に根深く残るパワーハラスメントの正体を、文化比較、統計データ、心理学的メカニズムの観点から深掘りします。単なる個人の資質の問題ではなく、日本の労働慣行や人口構造の変化、さらには脳の仕組みまでが複雑に絡み合っている現状を整理します。

津野香奈美 男性性（Masculinity） ホーフステードスコア 労災認定 一次予防 人口減少社会 ドーパミン 努力報酬不均衡モデル 相対的剥奪

📌 日本の文化とパワハラの親和性

  • 高い「男性性（Masculinity）」スコア
    • ホーフステードの文化比較研究において、日本は世界で断トツのトップクラス。
    • 成功、業績、勝利、力で打ち負かすことを極めて重視する価値観。
    • 「皆勤が当たり前」「高い地位への上昇が善」とされ、脱落者は「負け組」と見なされる文化。
  • 「和」の裏側にある排除の論理
    • 日本は同質性が高い組織（似たもの同士）が多く、異分子や邪魔者を排除する力が強く働きやすい。
    • 「みんなで仲良く」ではなく、ライバル同士が戦いながら、和を乱す者を攻撃する構造がある。

📌 統計に見るパワハラの深刻度

精神障害による労災認定において、パワハラは圧倒的な要因となっています。

項目 内容・数値
労災認定件数（2024年度） 精神障害全体で過去最高の1,000件を突破。
パワハラの割合 1,000件中224件が「上司からのパワハラ」で断トツの1位。
ハラスメント全体 カスハラ、セクハラ等を含めると、精神障害の労災認定の約4割がハラスメント関連。
統計の変遷 2020年の法制化で「パワハラ」項目が独立した途端、1位に踊り出た。

📌 企業の対策における「予防」の欠如

  • 義務化の限界：現在の法律で義務付けられているのは、相談窓口の設置や事実確認、再発防止策。これらは「起きた後」の対応。
  • 1次予防の重要性：そもそもパワハラが起こらない組織風土を作る「1次予防」まで到達している企業は少ない。

📌 人口減少社会における「パワハラ」の損失

  • 生産年齢人口の激減
    • 15歳〜64歳の人口は、過去30年で1,300万人減少。
    • 今後30年でさらに2,000万人減少することが確定している。
  • 「人を潰している場合ではない」という危機感
    • 労働力不足を外国人で補おうとしても、日本の「残業強制」「職務外の命令」といった労働慣行は海外ではパワハラと見なされ、トラブルの原因になる。
    • 今いる人材を一人でも長く、健康に働かせることが経営の最優先事項。

📌 なぜ個人はパワハラをしてしまうのか（心理・環境要因）

  • ストレスと不公平感
    • デマンドコントロールモデル：仕事の要求度が高く、裁量権（コントロール）が低い環境ではパワハラが発生しやすい。
    • 努力報酬不均衡モデル：頑張りに対して報酬（給与・評価）が低いと感じると、他者への攻撃性が高まる。
    • 相対的剥奪：自分にないものを他者が持っている（例：同じ仕事なのに給与が違う）と感じる剥奪感が攻撃に繋がる。
  • 「説教」という快楽
    • 相手のダメな点を指摘し、指導することでドーパミンが放出される。
    • 優越感や自己肯定感を得られるため、一種の中毒（依存症）状態になり、本人が自覚して止めるのは困難。
  • 職場の孤独
    • リモートワーク等で人と接する機会が減り、孤独を感じると疑心暗鬼になり、不安から攻撃的になりやすい。
    • 「社会的に排除されている」と感じる人は、他者に対して攻撃的に振る舞う傾向がある。

📌 結論と今後の展望

パワハラはなくならないのではなく、「なくすための仕組み」がまだ不十分である。経営層が人口減少のデータを直視し、パワハラを「組織の損失」として強く認識することが転換点となる。
