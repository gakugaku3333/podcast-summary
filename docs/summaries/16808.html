📌 航空業界の変革：ANAのルール変更と「アンバンドリング」

  • ANA国内線のルール変更（5月19日〜）：安い運賃では事前の座席指定ができなくなるなど、LCCに近い仕組みを導入。
  • 背景にあるコスト増：燃料費、人件費、空港利用料、整備費など、あらゆる販管費が上昇している。
  • 利用者の多様化：ビジネス客中心から、レジャー客やインバウンド客へと拡大。一律の「込み込み価格」が時代に合わなくなっている。
  • アンバンドリング（非セット化）の概念：
    項目 従来の「バンドリング」 これからの「アンバンドリング」
    価格構成 座席指定・荷物・変更権が全て込み 基本運賃を下げ、必要なオプションを個別購入
    ユーザー利点 何も考えなくて良いが、割高な場合も 不要なサービスを削り、安く乗る選択ができる
    航空側の利点 一律の利益率 需要に応じた柔軟な価格設定（収益性の改善）
  • 田中氏の視点：グローバルでは標準的な流れ。新しいルールに文句を言うより、素早く適応して使いこなす方が得策。

📌 田中啓の旅行哲学：予定調和を壊す「暮らすような旅」

  • 空港という場所への愛着：20代〜30代の頃は、用もないのに空港へ行き、離着陸を見てノスタルジーに浸っていた。
  • 空港の本屋での「ジャケ買い」：旅先で読む本は思い出と紐づくため、あえて現地で直感的に選ぶ。
  • 「確認作業」としての旅行からの脱却：
    • かつてはSNSやガイドブックで調べ尽くし、分刻みのスケジュールを組んでいた。
    • しかし、調べすぎると「実物を確認するだけ」になり、写真とのギャップにがっかりすることが多い。
    • 現在は「ほぼ何も調べない」スタイル。現地の治安情報だけ確認し、あとは「なすがまま」に過ごす。
  • トラブルを歓迎する：予定不調和な出来事やハプニングこそが、後々まで残る記憶になる。

📌 承認欲求との向き合い方

  • マズローの欲求5段階説をベースに考察。
  • 外発的承認：他者からの評価（名声、地位）。ラジオでの発信も「誰かに届けたい」という微かな承認欲求が含まれる。
  • 内発的承認（重要）：自分で自分を認めること。自己有能感や自尊心。
    • 田中氏は「毎朝のルーティンをこなす自分」を認めることで、自信を構築している。
  • SNSの批判への対処：
    「SNSでの『承認欲求お疲れ』といった攻撃的なコメントは、いわば便所の落書き。通り魔だと思って無視すればいい」

📌 食への異常なこだわりと「協調性の欠如」

  • 魚介類への極端な好み：
    • エビ、カニ、オイスター、ロブスターの「身」に興味がない。
    • 「ダシ」は好きだが、高級店でロブスターを出されてもテンションが上がらない。
    • 天ぷらも「野菜だけでいい」タイプ。
  • 一人飯の徹底：
    • 韓国旅行で全員が「ケジャン（カニ）」を食べようとしても、自分だけ離脱してサムギョプサルを食べに行く。
    • 限られた旅の食事回数を、興味のないものに費やしたくない。適当な嘘（現地の友人に会う等）をついてでも別行動をとる。
  • スイーツの変遷と「食感」のループ：
    • 流行は「柔らかい」と「硬い・もちもち」の間を行き来している。
    • 変遷例：ティラミス（ふわふわ）→ ナタデココ（コリコリ）→ タピオカ（もちもち）→ カヌレ（弾力）→ モッチュリン（最新トレンド）。
    • 田中氏は圧倒的に「噛み応え」重視。スープカレーもサラサラよりドロドロ派。
  • 極端な食べ方：
    • 一度ハマると毎日同じものを食べ続け、ある日突然「一生分の上限」を超えて二度と食べられなくなる。
    • 犠牲になった菓子：ロッテの「トッポ」「サシャ」。
  • 苦手な甘味：
    • 和菓子全般（豆、栗、かぼちゃを甘く煮たものが許せない）。
    • フルーツとケーキの合体（フルーツタルトはフルーツだけ先に食べる）。
    • ニューオータニ「スーパーショートケーキ」：1ピース5,940円。一度食べたが「別々でいいな」と再確認。
