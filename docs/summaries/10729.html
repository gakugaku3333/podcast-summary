油を変えれば寿命が伸びる？【食事編03 / あぶら】

「油＝体に悪い」という一括りのイメージを払拭し、脂質の種類（飽和・不飽和・トランス脂肪酸）ごとの健康への影響を解説。エビデンスに基づき、どの油を控え、どの油に置き換えるべきかという具体的な実践方法を提示するエピソード。

📌 脂質の全体像と分類

脂質は三大栄養素の一つであり、大きく分けて以下の3つの性質に分類される。

📌 飽和脂肪酸：摂りすぎ注意だが「適量」が鍵

  • 主な供給源（日本人の摂取源ランキング）
    順位 食品カテゴリ 割合
    1位肉類27%
    2位乳類（乳製品）25%
    3位油脂類9%
    4位穀類9%
    5位卵類8%
  • 健康への影響
    • 過剰摂取：心筋梗塞のリスクが上昇する。
    • 不足：日本の研究では、少なすぎると脳卒中のリスクが高まる可能性が示唆されている。
    • 目安：国立がん研究センターのデータでは、1日20g前後の摂取が心筋梗塞リスクを低く抑えられる。
  • ココナッツオイルの落とし穴
    • 「スーパーフード」として流行したが、科学的な厳密な定義はない（マーケティング用語の側面が強い）。
    • 実際は飽和脂肪酸が多く、LDLコレステロール値を上げるため、心臓協会が注意喚起を行うほど。
    • 「体に良いと思って積極的に摂るべき油」ではない。

📌 不飽和脂肪酸：積極的に選びたい「良い油」

  • 主な種類：オリーブオイル、ごま油、キャノーラ油（菜種油）、米油。
  • メリット：
    • 多く摂取している人ほど、動脈硬化や血栓の予防、慢性疾患のリスクが低い傾向にある。
    • 「油＝悪」ではなく、健康に良い影響を与える油として積極的に活用すべき。
  • 注意点：
    • 高カロリー：1gあたり9kcalあるため、健康に良いからといって飲み過ぎれば肥満や生活習慣病を招く。
    • 「これさえ摂れば健康」という過信は禁物。全体のバランスが重要。

📌 トランス脂肪酸：見えないリスクと食品表示

  • 特徴：不飽和脂肪酸を工業的に加工したもの。安価で保存性が高く、サクサクした食感を生む。
  • 日本人の主な摂取源：
    • お菓子類（18%）
    • パン類（18%）：特に菓子パンなどは砂糖だけでなくトランス脂肪酸も多い。
  • 健康リスク：動脈硬化、心筋梗塞などの冠動脈疾患リスクを上昇させる。
  • 日本の規制と表示の現状：
    • 平均摂取量が諸外国より少ないため、政府による義務化された規制はない。
    • 「植物性油脂」「植物性食用油」という表記に注意。これらが含まれる加工食品はトランス脂肪酸を含有している可能性が高い。
    • 自主的に「トランス脂肪酸ゼロ」「フリー」と表示している製品を選ぶのが賢明。

📌 実践的な「油の習慣」改善策

  • 「置き換え」の徹底
    • 肉（飽和脂肪酸）を減らし、魚（不飽和脂肪酸）を増やす。
    • バターの代わりにオリーブオイルを使う。
  • 「デフォルト設定」を決める
    • 洋風料理 → オリーブオイル、菜種油
    • 和風・中華 → ごま油
    • 揚げ物・汎用 → 米油
    • 家庭に置く油をこれらに固定することで、無意識に健康的な選択ができる。
  • 環境設計（物理的距離）
    • お菓子や菓子パンを買いだめしない。
    • 誘惑のある場所（コンビニやパン屋）を避けるルートで帰宅する。
  • ラーメンとの付き合い方
    • スープを飲み干さない（油の摂取量を物理的に減らす）。
    • ラーメンを食べた週は、他の食事で肉を避け魚中心にするなど、週単位で調整する。
