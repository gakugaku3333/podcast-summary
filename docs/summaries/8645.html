超実践的幸福論：タレント・壇蜜。幸福と距離を置く、独特の理由とは？（前編）

タレントの壇蜜氏をゲストに迎え、彼女のミステリアスな経歴や「幸福」に対する独特のスタンス、そして死生観について深掘りします。華やかな芸能界にいながら、なぜ彼女は「不幸をチラ見せする」ことを選ぶのか、その根源にある哲学が語られます。

壇蜜 遺体衛生保全士 エンバーミング ミイラ 末断意識 徳を積む 龍が如く SPA! 生きてさえいければいい

📌 壇蜜流「幸福」との距離感

  • 幸福をあえて避ける意識： 自身の仕事を「しなくてもいい苦労をしてお金を稼ぐこと」と定義。キラキラした幸せを発信することは自分の役割ではないと考えている。
  • 不幸のチラ見せ： 「ちょっと不幸なところをチラ見せしてご飯を食べている」という自覚があり、幸せを感じすぎないように意識している。
  • パブリックイメージの受容： ミステリアスで実体がつかめない存在として捉えられていることを、銭湯で服を脱ぎながら客観的に感じていた。

📌 芸能界入りの意外なプロセス

  • 始まりは「思い出作り」： 本来は葬儀社で遺体修復の仕事をする予定だったが、希望の枠が空かず断念。就職前の思い出として「素人グラビア」に応募したのがきっかけ。
  • ゲームからリアルへ： 「龍が如く」のキャバ嬢キャラ公募に合格し、そこからコンパニオン登録、グラビアへと道が開けた。
  • 執着のなさ： 芸能界で生き残るための強い野心があったわけではなく、「気づいたらわらじが置いてあったから履いた」という感覚。

📌 仕事観と「生きてさえいければいい」哲学

  • 生存が最優先： 「生きてさえいければいい、親に迷惑をかけなければいい」という低いハードルを設定することで、複雑な悩みや不安を解消している。
  • 自立への強い意志： 「自分のお金がないとダメ」という思考が強く、他人に頼りたくない。
  • 末断（まつだ）意識： 夫の清野とおる氏から教わった言葉。一族を最後に見取る、あるいは自分が最後であるという意識。これが辛い仕事や意地悪な扱いにも耐える原動力となっている。
  • 戻る場所がある強み： 大学病院の教授から「嫌われるのと好かれるのが半々だと長持ちする」「ダメなら戻ってくればいい」と言われたことが、芸能活動を続ける心の支えになった。

📌 遺体衛生保全（エンバーミング）への情熱

  • 14歳での「天啓」： 博物館でミイラを見た際、「自分ならこれを作れる」と直感的に確信した。
  • 遺体への抵抗感のなさ： 10代の頃から死や遺体に対して恐怖や抵抗がなく、むしろその修復に自分の適性を見出していた。
  • 仕事の醍醐味： 事故や病気で姿が変わってしまった遺体をきれいに修復し、家族が「寝ているみたいだ」と安心して触れてお別れできるようにすること。

📌 死生観と「徳」の積み方

項目 壇蜜氏の考え方
来世 あると思っている。ただし「次も人間になれると思うなよ」と自分を戒めている。
徳を積む行為 マンションのエントランスのゴミを拾うなど。穏やかな死を迎えるための準備。
物欲 非常に薄い。クローゼットにはジャージとワンピースしかない。
欲求の源泉 かつてはバイト代で何かを買う目的もあったが、現在は健康で仕事ができることに感謝している。

📌 精神的な安定と現状

  • 体調不良からの回復： 3年前に体調を崩して入院したが、現在は「やめない」と決めて少しずつリハビリ復帰している。
  • 再入院なしの3年間： 大きな揺り戻しなく過ごせている現状に満足しており、家族への感謝が強く、今は新たな欲求があまりない状態。
「生きてさえいければいいっていうところで設定すれば、小さい悩みは消えてくれるんです。」
