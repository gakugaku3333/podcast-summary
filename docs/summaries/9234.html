設立わずか4年で評価額が約6兆円に達したヒューマノイドスタートアップ「Figure AI」について、その異常な成長スピードの背景にあるCEOの戦略、技術的優位性、そしてビジネスモデルの特異性を深掘りします。

📌 Figure AIの驚異的な市場評価

  • 評価額 約6兆円： 創業わずか4年のスタートアップが、三菱重工（約12兆円）の半分、日立（約14兆円）に迫る評価を獲得。
  • 売上ほぼゼロでの高評価： 実用レベルでの商業利用はまだ先であり、BMWへのテスト納品数十台という段階での「期待値」による高騰。
  • 資金調達の推移：
    • 累計調達額：2,850億円。
    • 直近18ヶ月で評価額が15倍に跳ね上がった。
    • 他のヒューマノイドメーカーが3〜5倍の成長に留まる中、突出した数字。

📌 稀代の起業家 ブレット・アドコック（Brett Adcock）

  • バックグラウンド： 現在40歳。アメリカ中西部の3代目農家出身。16歳で起業。テック系ではなくビジネス・経営学出身。
  • シリアルアントレプレナーとしての実績：
    • 1社目（Vettery）： AIマッチングのHRテック。約165億円で売却し、個人資産を築く。
    • 2社目（Archer Aviation）： 空飛ぶタクシー（eVTOL）の開発。未経験から専門家300人にヒアリングし、3年で上場。2028年ロス五輪でのサービス提供を目指す。
  • Figure AIへの執念：
    • 2022年5月創業。自己資金150億円を投入。
    • VCに頼らず初期を自社資金で回すことで、経営権（株）を維持し、大胆な戦略を可能にした。

📌 なぜFigure AIは「勝てる」と思われているのか

  • フルスタック（垂直統合）開発：
    • AI、モーター、データ収集、製造のすべてを自社で行う（テスラと同様の戦略）。
    • 外部依存を減らすことで、利益幅の最大化とハード・ソフトの超密接な連携が可能。
  • VLA（Vision-Language-Action）モデル：
    • 視覚と言語から直接「関節の動き」を出力するモデル。
    • 従来のLLMを介した指示よりも圧倒的に高速で、APIコストも抑えられる。
    • OpenAIとの提携を解消し、自社開発に切り替えた点も強気な評価。
  • BMW工場での実証データ：
    • メタルシートの運搬・配置作業において、99%以上の成功率と5mm以下の誤差を達成。
    • 「実際に現場で使える」という実測値が投資家の信頼を生んでいる。
  • 圧倒的な量産計画：
    • 4年間で10万台の生産を宣言。一社で世界中のヒューマノイド累計台数を賄う規模を目指す。

📌 ヒューマノイド業界の3大派閥

派閥 脳（AI）の構造 特徴
NVIDIA派 2つの脳（思考＋反射） 経営企画（思考）と現場（実行）を分ける。多くの企業が採用。
Google Gemini派 1つのベテラン脳 経験豊富な社員のように、推論と行動を一つのモデルで行う。
Figure AI（独立） 自社専用VLA ハードとソフトを同時開発。手のひらにカメラを置くなど、独自の最適化。

📌 「データ」が次の主戦場になる

  • 学習データの価値： 頭の良さ（モデル）はコモディティ化し、実際の現場での「作業データ」をどれだけ独占できるかが勝負。
  • 意外なデータソースの可能性：
    • ポケモンGO： 人間の目線の高さで撮影された膨大な映像データは、ロボットのナビゲーションに極めて有用。
    • SNS（Xなど）： 人間らしい振る舞いや思考の学習データ。
    • 生体データ： 排泄物（うんち）のデータなど、一見不要に見えるものが生物学的データとしてビジネス化する兆し。
  • シミュレーションプラットフォーム： NVIDIAなどが提供する仮想空間での学習環境も、今後の投資対象として重要。

📌 市場の将来性とリスク

  • 2050年の予測： モルガン・スタンレーの試算では、ヒューマノイド市場は4,500兆円に達する。
  • シェア10%の価値： もしFigure AIが市場の10%を取れば売上450兆円。現在の6兆円という評価額は、その未来から見れば「格安」とも言える。
  • バブルの懸念： 実用化に失敗すれば、一気に評価が暴落し倒産するリスクも依然として存在する。
