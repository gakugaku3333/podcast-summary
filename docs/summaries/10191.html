子育てのラジオ「Teacher Teacher」：料理×人生相談

パーソナリティの二人が、リスナーから寄せられた人生相談に答えながら、限られた素材で複数の作り置き料理を完成させる新企画。 「地元の友達との価値観のズレ」という深い悩みに対し、料理の手を動かしながら独自の視点で切り込みます。

📌 人生相談：20年間フリーターの友人とどう付き合うか

  • 相談者（コウタロウさん・36歳）の悩み
    • 高校時代からの友人が16歳から20年間、同じバイト先でフリーターを続けている。
    • 正社員への誘いも「きつそう」「転勤があるかも」と断り、常に金欠状態。
    • 飲食店に行っても「お腹いっぱいだから」「水でいい」と何も注文せず、一緒に遊びづらい。
    • 友人に自信を持って何かにチャレンジしてほしいが、どう声をかければよいか。
  • 「資本主義の物差し」からの脱却
    • 成功の定義： 昇進や年収といった「資本主義のゲーム」の物差しで見ると友人が劣っているように見えるが、高校時代はそんな物差しはなかったはず。
    • 成熟の捉え方： 執着から解き放たれているという意味では、友人のほうが「悟り」に近い可能性もある。
    • 学びの機会： あえて自分も「水を頼む」など、友人のルール（資本主義外のルール）に合わせてみることで、当たり前の幸せを再発見できるかもしれない。
  • 存在価値（Being）と機能価値（Doing）
    • 機能価値： 「頭がいい」「仕事ができる」といった能力への称賛。こればかりを求めると、できない時にしんどくなる。
    • 存在価値： 「一緒にいて楽しい」「いてくれるだけで嬉しい」という全肯定。36歳で月1回会える友人は非常に貴重であり、まずはその存在自体への感謝を伝えるべき。
  • 具体的なアプローチ案
    • お金のかからない遊び： 将棋、ボードゲーム、散歩など、金銭的格差が気にならない活動を提案する。
    • 間接的な刺激： 本人へ直接「頑張れ」と言うと反発を招くため、モチベーションが上がる映画（ハイキュー!!、ビリギャル等）や講演会（植松努氏、成澤俊介氏等）に誘い、感想を共有する。

視点を変えることで「どうにかしたい対象」から「大切な友人」へ戻るプロセス

📌 料理企画：3つの素材で6品作る

基本素材： 豚ひき肉（600g）、卵（12個）、ブロッコリー

料理名 主な材料・ポイント
1. 味玉 ゆで卵を特製タレ（砂糖1:醤油4:水3:みりん2の比率）に漬け込む。
2. 卵と豚のそぼろ 豚ひき肉と卵をそれぞれ炒めて合わせる。お弁当の定番。
3. 肉味噌 ひき肉、味噌、にんにく、生姜、酒、みりん、醤油で味付け。万能おかず。
4. サルシッチャ ひき肉にスパイス（クミン、コリアンダー等）を混ぜ、ラップで成形して茹でる。
5. ブロッコリーと卵のマヨ和え 茹でたブロッコリーとゆで卵をマヨネーズ、塩、ブラックペッパーで和える。
6. ブロッコリーのナムル ブロッコリーをごま油、醤油、砂糖、すりごまで味付け。

📌 料理の知恵・テクニック

  • 味玉の黄金比： 砂糖1、醤油4、水3、みりん2（卵の個数に合わせて調整）。
  • ジップロックの空気抜き： ボウルに溜めた水の中に、食材を入れた袋を沈めていくことで、水圧を利用して空気を完璧に抜くことができる。
  • サルシッチャの簡略化： カルディの「イタリアンソーセージの素」などを使うと、ひき肉に混ぜるだけで本格的な味になる。
  • 料理のマインドフルネス： 仕事のことを忘れ、目の前の工程（卵を剥く、切るなど）に集中することで、忙しい日々の中でもリフレッシュできる。
  • スキルの習得： 「魚を捌けるようになる」「ソーセージを作れるようになる」といった新しい経験は、人生の質を一段階引き上げる。
